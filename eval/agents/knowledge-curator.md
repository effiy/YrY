# knowledge-curator 评测示例

真源：`.claude/agents/knowledge-curator.md`。

**评测约束**：从本次任务过程和产物中提取可复用知识，输出结构化经验记录。

---

## 成功判据（可观测）

| 维度 | 预期 |
|------|------|
| 过程分析 | 分析本次任务过程和产物 |
| 共性提取 | 识别并归纳跨 agent 的共性知识和模式 |
| 输出形式 | 输出结构化经验记录，不修改 agent 定义文件 |
| 可复用性 | 输出知识具有跨任务复用价值 |

---

## 负例（应判不达标）

- 未分析本次任务过程而仅凭印象总结。
- 修改 agent 定义文件或技能文件。
- 提取的知识过于具体，缺乏跨任务复用价值。
