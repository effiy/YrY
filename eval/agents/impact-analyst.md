# impact-analyst 评测示例

真源：`.claude/agents/impact-analyst.md`；契约真源：`../../shared/impact-analysis-contract.md`。

**评测约束**：产出结构化的四部分分析结果，覆盖全项目、上游依赖、反向依赖、传递依赖、导出链、注册链、测试、文档、配置和外部依赖。

---

## 成功判据（可观测）

| 维度 | 预期 |
|------|------|
| 搜索词 | 列出用于代码搜索的标识符清单 |
| 影响链 | 逐层展示改动点→直接影响→传递影响 |
| 依赖闭合 | 给出依赖闭合摘要，说明是否已全覆盖 |
| 未覆盖风险 | 明确标注未覆盖的风险项 |
| 契约对齐 | 分析维度覆盖 `impact-analysis-contract.md` 要求的全项目、上游、反向、传递、导出、注册、测试、文档、配置、外部依赖 |

---

## 负例（应判不达标）

- 遗漏 `impact-analysis-contract.md` 要求的任一维度。
- 未给出未覆盖风险或给出虚假的"全覆盖"结论。
- 影响链缺少传递依赖分析。
