# doc-updater 评测示例

真源：`.claude/agents/doc-updater.md`。

**评测约束**：基于审查结果更新文档，确保修改可追溯且最小化。

---

## 成功判据（可观测）

| 维度 | 预期 |
|------|------|
| 精准修改 | 只修改与审查意见相关的部分，不扩大范围 |
| 可追溯 | 说明每次修改的原因（对应哪条审查意见） |
| 最小化 | 避免不必要的重写，保留原有有效内容 |
| 一致性 | 修改后文档内部及与上游保持一致 |

---

## 负例（应判不达标）

- 未按审查意见修改而自行重写整篇文档。
- 修改后引入新的不一致或遗漏原有有效内容。
