# security-reviewer 评测示例

真源：`.claude/agents/security-reviewer.md`。

**评测约束**：审查代码和文档中的安全风险，输出安全问题列表与修复建议；必答问题须全部回答。

---

## 成功判据（可观测）

| 维度 | 预期 |
|------|------|
| 必答问题 | 完整回答 XSS、信息泄露、鉴权、CSRF 等必答问题 |
| 分级 | 安全问题按 P0/P1 分级 |
| 修复建议 | P0 项给出具体修复建议 |
| 覆盖 | 覆盖代码和文档中的安全风险 |

---

## 负例（应判不达标）

- 遗漏必答问题中的任一项。
- 发现安全风险但未给出修复建议。
- 未对安全问题进行 P0/P1 分级。
