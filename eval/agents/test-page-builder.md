# test-page-builder 评测示例

真源：`.claude/agents/test-page-builder.md`。

**评测约束**：构建可交互的测试原型页，用于快速验证 UI 方案。

---

## 成功判据（可观测）

| 维度 | 预期 |
|------|------|
| 可运行 | 原型页可直接在浏览器中打开并交互 |
| 场景覆盖 | 覆盖需求任务中的主要交互场景 |
| 状态完整 | 包含正常、异常、边界等关键状态 |
| 轻量 | 原型代码简洁，不引入不必要的依赖 |

---

## 负例（应判不达标）

- 原型页无法运行或缺少关键交互。
- 未覆盖需求任务中的主要场景。
- 原型代码过度工程化，引入不必要的复杂依赖。
