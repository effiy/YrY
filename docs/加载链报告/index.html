⚡ 加载链报告

YrY CDN 加载链稳定性指数 — 五步链路(shared/index.css → theme/index.css → 组件CSS → Vue 3 → 组件JS) · 三维评估(可靠性40%·容错性35%·回退策略25%)。数据源：cdn-summary/index.json + 客户端加载性能埋点
加载中... · 评级: A≥80 B≥60 C≥40 D<40 · 每 5min 自动刷新
加载链架构：YrY 页面从请求到完整渲染经过 5 步关键路径：shared/index.css(设计令牌基线)→theme/index.css(主题系统)→组件 CSS 批量加载(~20-30个)→Vue 3 运行时(jsDelivr CDN)→组件 JS 批量加载(~10-20个)。每步独立评分后加权综合。
评分模型：LSI = 可靠性×40% + 容错性×35% + 回退策略×25%。总加载目标≤800ms·首次渲染目标≤400ms·CDN 命中率目标≥95%。
可靠性(40%)：五步各步加载成功率·CDN 可用性(SLA≥99.9%)·DNS 解析耗时·HTTP 状态码分布·超时控制(5s)。
容错性(35%)：单步失败的优雅降级能力·页面仍可交互·视觉无严重破坏·超时重试策略(指数退避×3)·资源加载失败不阻塞后续步骤。
回退策略(25%)：fallback CSS 完备性·静态降级版本·备用 CDN 配置(unpkg+jsDelivr 双源)·本地缓存兜底(HTTP Cache+CDN Edge Cache)·字体 fallback 策略(system-ui→-apple-system→sans-serif)。
性能目标：FCP≤500ms · LCP≤1s · CLS≤0.1 · FID≤50ms · TBT≤200ms · TTFB≤200ms · Speed Index≤1.5s
