自动巡检中心

12 技能定期巡检，由 Cron 定时任务按预设间隔触发执行。每轮执行生成自循环报告 HTML，rui-bot loop-report 汇总并推送企微通知。关联维度数据输入 D0-D8 诊断引擎，驱动持续自改进闭环。
— 份报告 · 覆盖 — 个技能模块 · 自举引导模式
巡检机制：Cron 定时任务按预设间隔触发技能执行 → 每轮生成自循环报告 HTML → rui-bot loop-report 汇总统计 → 企微 Webhook 推送通知
报告位置：docs/自循环报告/ · 汇总工具：node skills/rui-bot/lib/loop-report.mjs · 通知渠道：企业微信 Webhook
诊断关联：巡检异常按严重程度触发对应诊断 — fail → D0 基线偏离 · warn → D2 质量退化 · 连续 3 次 warn → 自动升级为 fail
数据新鲜度：报告文件直接从文件系统读取，reports.json 由 loop-report 在每次巡检完成后自动更新
报告质量分层：pass (✅ 全绿) → 正常 · warn (⚠ 发现项 ≥1) → 需关注 · fail (🚫 异常 ≥1) → 需立即处理
