Panel hub 心智模型: Cron 定时触发 → 技能执行 → 通知产出 → 🔔 通知面板展示 → 数据写入 .memory/ → 🧬 自改进分析消费。四个面板按钮是这整条数据链的入口。

⏰ 调度面板

  • 数据源: .claude/scheduled_tasks.json
  • 展示: cron 表达式 · 描述 · 活跃状态 · 下次触发
  • 操作: 查看全部定时任务 · 按状态筛选

🔔 通知面板

  • 数据源: 健康报告 + 自循环报告 + 趋势报告
  • 展示: 按日期展示最新通知 · 按类型筛选
  • 操作: 评分趋势追踪 · 诊断触发追踪

🧬 自改进面板

  • 数据源: .memory/health-trend.jsonl + summary.json
  • 展示: 日/周/月/全景四视角 · D0-D7 诊断
  • 操作: 等级分布 · 分支健康对比 · 趋势图表
