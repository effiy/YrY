自动化生成管线 — 演示

/rui-html 首页 · 3 触发路径 · 5 步生成流程

🚀 手动触发

$ /rui-html 首页
→ 扫描项目资产 → 渲染六层 HTML → 写入 docs/index.html

🔄 强制覆盖

$ /rui-html 首页 --force
→ 忽略已有文件 → 全量重新生成 → 更新版本号

👁️ 预览模式

$ /rui-html 首页 --dry-run
→ 采集数据 → 渲染 HTML → 输出到 stdout（不写入文件）

🏗️ init 集成

$ /rui-init arch 步骤
→ 项目初始化时自动生成首页基线 → 建立文档中心入口
