资产扫描策略

FP1
  • 遍历目录 — skills/ · agents/ · rules/ · docs/故事任务面板/
  • 提取 frontmatter — name · description · version · tags
  • 自动发现 — 新模块自动纳入，非硬编码列表

元数据编目

FP4
  • 信息卡模板 — 名称 + 描述 + 版本 + 标签 + 链接 + 行数

FP3
