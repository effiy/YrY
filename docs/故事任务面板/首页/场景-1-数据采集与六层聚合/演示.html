数据采集与六层聚合 — 演示

5 条验证命令 · 可独立运行 · 证据等级 A

📦 Step 1: 扫描技能目录

$ ls skills/*/SKILL.md | wc -l
19
遍历 skills/ 目录，统计全部 SKILL.md 文件。结果 = 首页 Layer 2 的技能数。

🤖 Step 2: 扫描 Agent 目录

$ ls agents/*.md | grep -v AGENT.md | wc -l
9
统计 Agent 角色规约（排除 AGENT.md 拓扑总纲）。结果 = 首页 Layer 5 的 Agent 数。

📜 Step 3: 扫描规则目录

$ ls rules/*.md | wc -l
16
统计 rules/ 目录下的全部 Markdown 规则文件。结果 = 首页 Layer 5 的规则数。

📖 Step 4: 扫描故事目录

$ ls -d docs/故事任务面板/*/ | wc -l
5
统计故事任务面板下的故事目录数。结果 = 首页 Layer 3 的故事数。

🎬 Step 5: 扫描场景目录

$ find docs/故事任务面板 -name 'index.md' -path '*/场景-*' | wc -l
29
递归查找全部场景 index.md 文件。结果 = 首页 Layer 4 的场景数。
