初始化项目基线
运行 /rui init 建立项目骨架。detect → explore → generate → arch → setup → verify，自动识别技能、Agent 和规则的模块结构。
/rui init📋
查看模块全景
阅读 CLAUDE.md 中的项目画像和角色拓扑。AGENT.md 定义了 8 个 Agent 的职责分配和协作关系。
cat CLAUDE.md agents/AGENT.md📋
探查技能模块入口
每个技能目录下的 SKILL.md 是模块规约入口。rui 管线串行调用 6 个技能形成完整 SDLC。
ls skills/*/SKILL.md📋
验证 Agent 职责边界
阅读每个 Agent 的 .md 文件确认职责范围。pm 决策 → architect 设计 → coder 实现 → tester 验证 → security 审计 → reporter 报告。
ls agents/*.md📋
检查规则覆盖
rules/ 目录定义了管线纪律、安全护栏、文档约束和自改进规范。每条规则对应一个治理维度。
ls rules/*.md📋
