🧩 🧩 场景-1-模块定位 · 交互示例

三十秒内定位任一模块（技能/Agent/规则）的入口、依赖、消费者和契约

🚶 操作步骤

定位 skill 入口
执行 /rui help 查看所有可用命令，根据需求选择对应 skill
node skills/rui/help.mjs📋
查看依赖关系
打开 SKILL.md 查看 "依赖" 章节，了解上下游关系
grep -r "module-name" skills/📋
反向查找消费者
搜索所有引用该模块的文件，构建消费者列表
grep -r "skill-name" --include="*.md" --include="*.mjs"📋
输出定位报告
汇总入口、依赖、消费者信息，生成定位摘要

💡 预期输出

模块入口路径: skills/xxx/SKILL.md
依赖数: 3 (直接) + 5 (间接)
消费者: 2 个 Agent + 1 个规则文件
