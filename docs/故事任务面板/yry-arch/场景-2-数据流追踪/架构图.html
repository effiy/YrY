🔄 数据流

/rui code → 分支检查 → Gate A → coder 实现 → tester 验证 → code-reviewer → security → Gate B → 交付

📋 关键决策点

• Gate A 门禁：测试先行，RED 阶段必须失败非语法错误
• 逐模块策略：P0 不清零不进下一模块
• Gate B 门禁：验证轮数 ≤ 2，超限触发自改进
• 交付收口：日志追加 → 文档同步 → 企微通知
