📋 rui-story · 故事面板管理

远端查询 · 同步 · 健康检查 — 1-5 数字键切换 Tab · 悬停代码复制

🚶 演练 📊 概览 ⚡ 命令 📁 文件 🧪 测试
状态概览
远端查询所有故事 → 按状态分组统计 + 最近 5 个活跃故事。数据源为远端 API，不读本地文件系统。
/rui-story📋
进度全景
远端查询 → 所有故事详细表格。含类型推断 + git 分支关联，方便追踪每个故事的当前状态。
/rui-story list📋
从远端同步故事文档
远端 → 本地覆盖（委托 rui-import mode=pull）。可指定故事名同步单个故事，或全部同步。
/rui-story sync user-login📋
健康检查
检查 API 凭据有效性、远端可达性、项目配置完整性、数据格式合规性。不修改任何数据。
/rui-story health📋
删除故事目录
仅删除本地故事目录（先展示内容后确认）。不影响远端数据，可重新 sync 恢复。
/rui-story remove user-login📋
