📈 rui-trends · 技术趋势

代码库健康分析 + 多源技术趋势发现 — 1-5 数字键切换 Tab · 悬停代码复制

🚶 演练 📊 概览 ⚡ 命令 📁 文件 🧪 测试
基础统计
快速了解项目规模：总文件数、各模块文件分布、行数统计、技术栈概况。
/rui-trends📋
模块规模概览
按模块统计文件数、行数、最大文件。识别膨胀模块和需要拆分的代码区域。
/rui-trends modules📋
大文件检测
列出 ≥500 行的文件，支持 --threshold 自定义阈值。识别需要重构的巨型文件。
/rui-trends large-files --threshold 300📋
Git 变更热点
近 30 天高频变更文件排行。识别不稳定模块和需要加固的代码区域。支持 --since 和 --top。
/rui-trends hotspots --since 2026-01-01 --top 10📋
组件审计
统计 zk- 前缀组件的引用频次，识别未使用和过度使用的组件。
/rui-trends components📋
一键全量扫描
modules + large-files + hotspots + components 四合一，输出完整健康报告。
/rui-trends all📋
