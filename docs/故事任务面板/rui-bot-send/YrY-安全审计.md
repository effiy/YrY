## §1 资产识别

| 资产 | 敏感度 | 存储 | 传输 |
|------|:--:|------|------|
| API_X_TOKEN | 高 | 环境变量 | HTTP Header → 企微 API |
| 消息内容 | 中 | CLI 参数→内存→日志文件 | HTTP Body → 企微 API |
| 故事元数据 | 低 | 日志文件 | — |

---

## §2 STRIDE 威胁建模

| 类别 | 威胁 | 可能性 | 缓解 |
|:--:|------|:--:|------|
| S | 未授权发送通知 | L | Token 鉴权 |
| T | 消息内容被篡改 | L | HTTPS 传输 |
| I | Token 在日志中泄露 | L | Token 仅环境变量，不写入日志 |
| I | 敏感内容通过通知泄露 | L | 内容由管线控制，非用户自由输入 |
| D | API 不可达阻塞管线 | L | 降级不阻断，最多 3 次重试 |
| E | 消息注入恶意内容 | L | 内容为管线生成的结构化字段 |
