# 通知日志

> rui-bot 企微消息推送通知记录。每次推送后自动追加条目。

## 格式

| 字段 | 说明 |
|------|------|
| 时间 | ISO 8601 格式，推送发生时刻 |
| 故事 | 关联的故事名称（kebab-case） |
| Bot Key | 企微 webhook bot key（脱敏） |
| 状态 | `sent` / `failed` / `skipped` |
| 消息摘要 | 推送内容的简短摘要 |

## 日志

| 时间 | 故事 | 状态 | 摘要 |
|------|------|------|------|
| — | — | — | 暂无记录 |

> 条目由 `node skills/rui-bot/send.mjs` 自动追加。手动触发命令：`/rui-bot send --story=<name> --status=<s>`。
