查看包完整信息
获取版本历史、许可证、维护者、依赖关系、仓库链接。
/rui-npm info lodash📋
💡 说明: info 展示包的完整元数据，用于依赖审计决策。重点关注许可证合规性和已知漏洞数。
✅ 结果: lodash@4.17.22 — 许可证: MIT ✅ / 维护者: jdalton + 5 人 / 143 版本 / 0 运行时依赖 / 周下载量: 38.5M。
安全漏洞审计
检查所有依赖的已知 CVE 漏洞，按严重级别分组展示。
/rui-npm audit📋
💡 说明: npm audit 对照 GitHub Advisory Database 检查依赖漏洞。High/Critical = P0 需立即修复。
✅ 结果: 🔒 npm audit 扫描 15 个包 — 🔴 Critical: 0 / 🟠 High: 0 / 🟡 Moderate: 1 → 建议升级到最新版修复。
更新有漏洞的包
升级到兼容的最新版本，展示版本变更。
/rui-npm update lodash📋
💡 说明: update 在发现漏洞后的标准操作是升级到修复版本。升级后必须重新运行 audit 确认漏洞清零。
✅ 结果: ⬆ lodash: 4.17.22 → 4.17.23 (安全补丁) — audit 重新扫描: ✅ 0 漏洞。
卸载不再需要的包
从 package.json 和 node_modules 同时移除。
/rui-npm uninstall moment📋
💡 说明: uninstall 同时从 package.json 和 node_modules 移除。moment.js 已被 Day.js/luxon 等替代方案取代。
✅ 结果: 🗑 moment@2.30.1 已卸载 — dependencies 已移除 / 释放: 4.2MB / 依赖树: 15→14 个包。
