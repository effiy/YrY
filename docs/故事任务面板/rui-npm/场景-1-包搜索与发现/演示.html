搜索 react 相关包
按关键词搜索 npm registry，结果按周下载量降序排列，展示前 20 条。
✅ 结果: 搜索返回 20 条结果（按周下载量降序）— #1 react(19.2M/周) / #2 react-dom(17.8M/周) / #3 react-scripts(8.5M/周) … 每项显示包名、描述、版本、周下载量、许可证。
💡 说明: search 默认按 popularity 排序，展示前 20 条。结果缓存在本地 15 分钟，相同关键词重复搜索不消耗 API 配额。
/rui-npm search react📋
查看 react 详细信息
确认许可证、维护者、依赖健康度，确保项目兼容。
✅ 结果: react@19.2.0 — 许可证: MIT ✅ / 维护者: 6 人 / 依赖: 0 运行时依赖 / 周下载量: 19.2M / 仓库: github.com/facebook/react。
💡 说明: info 展示包的完整元数据，用于安装前进行许可证合规检查和依赖健康度评估。
/rui-npm info react📋
安装 react 到项目
安装到当前项目 dependencies，自动写入 package.json。
✅ 结果: ✅ react@19.2.0 已安装到 dependencies — package.json 已安装 / 安装耗时: 2.3s。
💡 说明: install 默认安装最新稳定版并写入 dependencies。安装后建议运行 /rui-npm list 确认版本。
/rui-npm install react📋
获取 react CDN 引用地址
查询 react 在 unpkg / jsDelivr / esm.sh 的 CDN 引用地址，直接在浏览器中通过 <script> 或 import 使用。
✅ 结果: 三路 CDN URL：unpkg → https://unpkg.com/react@19.2.0/ / jsDelivr → https://cdn.jsdelivr.net/npm/react@19.2.0/ / esm.sh → https://esm.sh/react@19.2.0。
💡 说明: cdn 命令自动获取包的最新版本号，生成版本化 URL。也可指定版本如 /rui-npm cdn react@18.2.0。支持 --json 标志输出机器可读格式。
/rui-npm cdn react📋
获取指定版本 CDN 地址
指定 react@18.2.0 版本，URL 中含精确版本号，确保生产环境版本锁定。
✅ 结果: 三条 URL 均含 @18.2.0 版本号 — 可直接复制到 HTML 的 <script src="..."> 标签中使用。
💡 说明: 生产环境建议指定精确版本号避免 CDN 缓存导致的不一致。esm.sh 支持直接 import，适合现代 ES module 项目。
/rui-npm cdn react@18.2.0📋

搜索
/rui-npm search react
按关键词搜索 npm registry，结果按周下载量降序，展示前 20 条。
/rui-npm search react📋
JSON 模式
/rui-npm search react --json
输出合法 JSON 数组，每元素含 name/version/desc/downloads。
/rui-npm search react --json📋
限制结果
/rui-npm search react --limit 5
返回最多 5 条结果，适合快速扫描。
/rui-npm search react --limit 5📋
Scope 包
/rui-npm search @types/node
支持 scope 包搜索（@scope/pkg 格式）。
/rui-npm search @types/node📋
CDN 引用
/rui-npm cdn react
获取 react 的三路 CDN 引用地址（unpkg/jsDelivr/esm.sh），使用最新版本。
/rui-npm cdn react📋
CDN 版本化
/rui-npm cdn react@18.2.0
指定精确版本号，URL 中含版本号确保生产环境锁定。
/rui-npm cdn react@18.2.0📋
