场景-1 · 包搜索与发现 + CDN 引用 — 架构图

npm search + cdn 命令流向：用户输入 → registry 查询/验证 → JSON 解析 → 表格输出 / CDN 三源 URL

命令流

  • search — 关键词搜索 npm registry
  • cdn — 查看包 CDN 引用地址（三源）
  • 校验关键词非空 / 包名非空 → 友好提示
  • JSON 解析 → 按 downloads 降序
  • 双格式输出：表格 | --json

错误处理

  • 关键词为空 → 提示用法 + 示例
  • registry 不可达 → npmjs.com URL
  • 包不存在 (cdn) → 明确报错
  • JSON 解析失败 → 明确报错不崩溃

输出格式

  • search 表格：名称/描述/版本/下载量/更新时间
  • cdn 表格：CDN 名称 / URL（unpkg/jsDelivr/esm.sh）
  • JSON：--json 标志输出原始数据
  • 按周下载量降序，top 20 结果
