rui-npm 知识图谱

v1.2.0 · 2026-06-06 · 10 步骤 · 3 数据域 · 5 数据流 · 5 连接边

🌐 数据域

npm Registry 外部 公共 npm 包注册中心，提供包搜索、元数据查询、包上传功能
本地项目 内部 用户当前工作目录，包含 package.json 和 node_modules
本地文件系统 内部 用户本地的 JavaScript 文件和目录，作为 publish 的输入源

📊 步骤（10 个子命令）

search按关键词搜索 npm registry
cdn查看包在 unpkg/jsDelivr/esm.sh 的 CDN 引用地址
install安装包到当前项目
update更新已安装包到最新兼容版本
list列出当前项目已安装的依赖
info查看包的完整元数据
uninstall从当前项目移除包
publish发布本地文件/目录为 npm 包
npx通过 npx 直接执行包
audit审计依赖的已知安全漏洞

🔀 数据流

包搜索流 关键词 → registry 搜索 → 解析排序 → 结构化输出 步骤: search
CDN 引用发现流 包名+版本 → npm view 验证 → 三路 CDN URL 生成 → 表格/JSON 输出 步骤: cdn
包安装流 指定包名 → 前置检查 → npm install → 版本确认 步骤: install · update · list
本地发布流 文件/目录 → 登录检查 → package.json → npm publish → 即时可用 步骤: publish · npx
审计清理流 元数据查询 → 安全漏洞检测 → 卸载清理 步骤: info · audit · uninstall

🔗 连接边

本地项目 → npm Registry 查询/下载（搜索·CDN·安装·审计流）
本地文件系统 → npm Registry 上传发布（本地发布流）
npm Registry → 本地项目 包安装（安装流）
npm Registry → 本地文件系统 npx 拉取执行（发布流）
