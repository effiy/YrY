安装 lodash 到项目
安装到 dependencies，自动写入 package.json，确认安装版本。
✅ 结果: ✅ lodash@4.17.21 已安装 — package.json dependencies 新增 "lodash": "^4.17.21" / node_modules/lodash/ 5.8MB。
💡 说明: @4.17.21 指定精确版本安装，^ 前缀表示兼容范围。生产依赖写入 dependencies，运行时可用。
/rui-npm install lodash@4.17.21📋
安装 devDependency
安装开发依赖，如 prettier、eslint 等工具链。
✅ 结果: ✅ prettier@3.6.2 已安装到 devDependencies — 12 个子依赖 / 安装耗时: 5.8s。
💡 说明: --dev 标志将包安装到 devDependencies（开发/构建依赖），生产环境不会安装。
/rui-npm install prettier --dev📋
更新 lodash 到最新兼容版
记录更新前版本，展示版本变更。
✅ 结果: ⬆️ lodash: 4.17.21 → 4.17.22 (补丁更新) — 修复 2 个 Bug / 无破坏性变更。
💡 说明: update 默认升级到兼容范围内最新版本。补丁版本（4.17.x）只包含 bug 修复。
/rui-npm update lodash📋
查看当前依赖树
列出所有直接依赖及版本。
✅ 结果: 📋 直接依赖 (3 个) — lodash@4.17.22 / prettier@3.6.2 / 总计: 3 + 12 = 15 个包。
💡 说明: --depth 0 仅列出直接依赖，--depth 1 增加一层子依赖。定期运行 list 检查依赖树健康度。
/rui-npm list --depth 0📋
