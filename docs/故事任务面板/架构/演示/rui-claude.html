管理所有 .claude/ 目录 — 1-5 数字键切换 Tab · 悬停代码复制

🚶 演练 📊 概览 ⚡ 命令 📁 文件 🧪 测试
推荐任务
五层评分排序（L0 时间 / L1 依赖 / L2 风险 / L3 覆盖 / L4 质量），输出优先级排序的 .claude/ 维护建议。
/rui-claude📋
覆盖式同步远端配置
远端 API → rui-import pull 覆盖本地 .claude/ 目录。适合跨设备同步或初始化新环境。
/rui-claude sync📋
插件升级
git pull 最新 YrY → 清除插件缓存 → sync 远端 .claude/ 双刷新。保持插件和配置同步最新。
/rui-claude update📋
健康度分析
三节复盘：配置结构完整性 → 健康度评分 → 改进项推荐。支持 --json 格式输出。
/rui-claude retro📋
需求管线修改配置
走 /rui code 流程修改 .claude/ 配置，有分支隔离和 Gate A/B 保护。适合大规模配置变更。
/rui-claude 添加新的安全护栏规则📋
全量扫描
跨项目扫描所有 .claude/ 目录状态，输出汇总报告。适合多项目环境治理。
/rui-claude scan --all📋
