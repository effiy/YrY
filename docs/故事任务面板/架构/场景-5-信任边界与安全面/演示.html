演示 3: 分支隔离阻断

在 main 分支尝试写入操作，预期被 branch-check.mjs 阻断

$ git branch --show-current
main
$ node skills/rui/branch-check.mjs
BLOCKED: bad-branch
  Current branch: main
  Required pattern: feat/<name>
  Reason: Edit/Write operations require feature branch isolation.
  Action: git checkout -b feat/<name>

[rui-import] This is a non-blocking degradation.
[rui-import] Set API_X_TOKEN to enable document sync.
