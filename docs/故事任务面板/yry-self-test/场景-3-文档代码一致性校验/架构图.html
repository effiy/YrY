YrY Self-Test — 场景 3: 文档代码一致性校验

文档代码漂移检测 · 4 层纵深校验 · 引用可达 · 实施匹配 · 测试对齐 · 图谱有效 · 真相对齐

🚫 阻断层 (L1)

  • • L1: 文件引用可达性
  • • 包含: Markdown 相对链接 + 图谱 source/target
  • • 任何断裂即阻断
  • • 报告含: 来源 + 目标 + 断裂原因
  • • 精确到文件行号或章节

⚠ 警告层 (L2–L4)

  • • L2: 实施报告 vs 版本控制日志
  • • L3: 测试报告 vs 实际测试输出
  • • L4: 知识图谱 vs 文件系统
  • • 一层失败不阻止其余层
  • • 可指定单故事定向检查

📊 差距量化

  • • 断裂引用数: N 处
  • • 实施报告差异: M 处
  • • 测试报告偏差: K 处
  • • 图谱失效节点: P 个
  • • 非 “有/无” 二元判断
