零外部依赖内联 JS

事件契约

统一
  • 所有组件使用 标准 DOM 事件（click/change/input）
  • 键盘事件仅处理 非 input 聚焦 状态，避免误触发
  • 事件处理函数 纯同步，无 Promise/async
同步DOM 原生

状态流向

单向
  • 用户操作 → 事件 → 状态更新 → DOM 渲染 → localStorage
  • 页面加载 localStorage → 恢复状态 → DOM 初始渲染
  • 无全局 store，每个组件独立管理自身状态
单向数据流
