💡 模板架构常见误区 避坑指南

⚠️
不要打乱 head 中 link 顺序
yry-checklist.css 必须在 theme/index.css 之后加载，否则 --yry-* 变量在 alias 时尚未定义，整个页面颜色变黑。CDN 顺序：shared → theme → yry-checklist → shared.js。
🚫
不要把 {{CDN_DEPTH}} 写成硬编码 ../
场景目录深度变化时，硬编码路径立即 404。让模板引擎按 __dirname.split('/').length 动态计算（{{CDN_DEPTH}} → ../../../../）。
✅
善用 @layer 显式优先级
yry-checklist.css 显式声明 5 层 @layer：reset → base → components → utilities → overrides。组件类在 components 层定义，utilities 层可被覆盖。新增组件时严格落层。

🐞 模板架构排错速查 高频问题

🔧
页面打开后无样式（FOUC）
症状：HTML 裸奔 · 控制台 404。CDN 顺序错位或路径错。检查项：
curl -I ../../../../cdn/shared/index.css
⏱
--yry-* 变量未定义导致颜色变黑
症状：DevTools Computed 中 --yry-bg 值为空。yry-checklist.css 加载在 theme/index.css 之前。修复：
mv <head> 中两个 link 标签的顺序
🔒
{{CDN_DEPTH}} 漏替换 · 链接全 404
症状：所有 CSS/JS 链接变成 {{CDN_DEPTH}}/cdn/xxx。模板渲染器未跑。
node lib/template-engine/render.mjs --strict
🧭
响应式断点不生效
症状：viewport 缩到 500px 卡片仍溢出。@media 查询缺失或断点值错。查看：
grep -n "@media" cdn/yry-checklist/index.css
📐
@layer 顺序被破坏 · 字号回退
症状：.kpi-card 字号变 16px。有人把 components 层放到了 reset 之前。修复：
cat cdn/yry-checklist/index.css | head -5
🔄
修改 yry-checklist.css 后浏览器无变化
症状：明明改过了，但浏览器还显示旧样式。可能是 HTTP 缓存。
Ctrl+Shift+R 强刷 / 或加 ?v=2 后缀
