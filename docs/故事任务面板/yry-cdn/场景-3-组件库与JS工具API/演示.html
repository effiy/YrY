测试 YrY.toast() 通知系统
在 Console 中调用 YrY.toast('message') 观察底部居中通知的显示与自动消失
✅ 结果: toast 在页面底部居中显示，1.8s 后 CSS transition 淡出。连续调用 3 次 toast，前一条定时器被清除，始终显示最新消息。
💡 说明: YrY.toast 内部使用 _toastTimer 管理定时器生命周期，clearTimeout 确保没有多个 toast 竞态。DOM 容器在首次调用时懒创建。
YrY.toast('Hello CDN!')📋
测试 YrY.copyCmd() 剪贴板复制
点击页面中任意 📋 按钮，观察按钮状态变化：📋 → ✅ → 📋
✅ 结果: 点击按钮 → navigator.clipboard.writeText() → 按钮文案变为 ✅ → 1.5s 后恢复为 📋。剪贴板内容可在任意应用粘贴验证。
💡 说明: copyCmd 封装了"复制→反馈→恢复"三步交互。使用 Clipboard API（需 HTTPS），fallback 到 document.execCommand('copy')。
// 点击本页任意 📋 按钮即可测试📋
测试 YrY.switchPanel() 面板切换
点击不同 Tab 观察面板切换动画和 tab 高亮状态变化
✅ 结果: Tab 点击 → switchPanel('commands') → 旧面板 fadeOut/新面板 fadeInUp 动画 → Tab 下划线切换为 accent 色。快捷键 1-5 也可触发。
💡 说明: switchPanel 通过 data-panel 属性关联 tab 和 panel。所有面板默认 display:none，仅当前面板 display:block + fadeInUp 动画。
// 控制台测试面板切换
YrY.switchPanel('commands')  // 切换到命令 Tab📋
测试 Suite Toggle 折叠组件
在审查页面的证据链中点击 suite 标题展开/折叠测试套件
✅ 结果: 点击 suite-head → classList.toggle('open') → 箭头旋转 90° (expand) / 0° (collapse) → body 区域 slideDown/slideUp 动画。
💡 说明: Suite toggle 使用事件委托——click 事件绑在父容器上，通过 e.target.closest('.s-head') 识别点击目标。支持嵌套 suite（内层 suite 不冒泡）。
// 打开审查.html → 点击"📎 审查证据清单"📋
查看工具函数 (fmtDur / esc / clipboardWrite)
在 Console 中测试 fmtDur 时间格式化、esc XSS 防注入、clipboardWrite 剪贴板写入
✅ 结果: fmtDur(1234) → "1.2s"。esc('') → 转义后安全字符串。clipboardWrite 复制到系统剪贴板。
💡 说明: fmtDur 自动选择 ms/s 单位。esc 转义 & < > " 四个字符防止 XSS。这些工具函数被 toast/copyCmd 等公开 API 内部调用。
YrY.fmtDur(1234)   // → "1.2s"
YrY.esc('')    // → "<xss>"📋
验证 55 消费者页面集成
grep 统计所有引用 shared.js 的 HTML 页面数量和分布
✅ 结果: grep -r "shared.js" docs/ 返回 55 个页面引用。Cat A（架构/知识图谱）2 页面 + Cat B（审查/测试/演示/计划）29 页面 + 额外页面 24 个。
💡 说明: 所有 55 个页面通过同一个 shared.js 获取 YrY API。IIFE 模式确保每个页面的 window.YrY 实例相互独立，无跨页面状态污染。
grep -r "shared.js" docs/故事任务面板/ | wc -l  # → 55📋
