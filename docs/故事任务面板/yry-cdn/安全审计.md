## STRIDE 威胁矩阵

### S — Spoofing (身份伪装)

| # | 威胁 | 可能性 | 影响 | 缓解 | 状态 |
|---|------|--------|------|------|------|
| S1 | 恶意文件替换 CDN 资源 | L | H | CDN 文件在 git 仓库中管理，修改需通过 git 提交 + code review | ✅ 已受控 |
| S2 | npm 包被伪装发布 | L | H | npm 双因子认证 + package.json 声明 GitHub 仓库来源 | ⚠️ 需确认 2FA 已启用 |
| S3 | 伪造 Google Fonts 响应注入恶意字体 | L | L | Google Fonts 仅加载字体文件（woff2），不含可执行代码 | ✅ 已受控 |

### T — Tampering (数据篡改)

| # | 威胁 | 可能性 | 影响 | 缓解 | 状态 |
|---|------|--------|------|------|------|
| T1 | 中间人篡改 CDN CSS/JS（网络传输） | L | H | 资源通过本地文件系统加载，不走网络（页面在本地打开） | ✅ 不适用 |
| T2 | 恶意页面覆盖 `window.YrY` 对象 | L | M | YrY 为全局对象，后加载的脚本可覆盖；但所有页面由同一可信源生成 | ⚠️ 低风险 |
| T3 | npm publish 时注入恶意文件 | L | H | package.json `files` 白名单限制发布文件（5 个已知文件） | ✅ 已受控 |

### R — Repudiation (不可否认性)

| # | 威胁 | 可能性 | 影响 | 缓解 | 状态 |
|---|------|--------|------|------|------|
| R1 | CDN 变更无法追溯 | L | M | git commit + tag 记录每次变更，版本号链可完整回溯 | ✅ 已受控 |
| R2 | npm 发布操作无审计记录 | L | L | npm registry 保留发布时间戳和发布者信息 | ✅ 平台提供 |
| R3 | 消费者页面迁移无记录 | M | L | git diff 记录迁移变更，commit message 标注迁移页面 | ⚠️ 依赖开发者纪律 |

### I — Information Disclosure (信息泄露)

| # | 威胁 | 可能性 | 影响 | 缓解 | 状态 |
|---|------|--------|------|------|------|
| I1 | Google Fonts 请求泄露用户浏览行为 | H | L | 浏览器向 fonts.googleapis.com 发送 Referer 头；仅加载字体 CSS，不加载 JS 或 tracking pixel | ⚠️ 隐私影响低但存在 |
| I2 | CSS/JS 源码泄露项目架构信息 | L | L | 静态资源不包含密钥/Token/敏感逻辑；项目为开源（MIT） | ✅ 无敏感信息 |
| I3 | npm package.json 泄露仓库 URL | L | L | 项目为公开仓库（GitHub），仓库 URL 非敏感信息 | ✅ 无影响 |
| I4 | 页面路径结构通过相对路径暴露 | L | L | `../../../../cdn/` 路径暴露目录层级深度；项目为内部工具 | ✅ 无影响 |

### D — Denial of Service (拒绝服务)

| # | 威胁 | 可能性 | 影响 | 缓解 | 状态 |
|---|------|--------|------|------|------|
| D1 | Google Fonts CDN 不可达导致字体加载超时 | H | L | 浏览器超时后回退系统等宽字体（monospace），页面内容不受影响 | ✅ 已受控 |
| D2 | shared.js 加载失败导致所有交互不可用 | L | M | 文件从本地加载，不受网络影响；HTML/CSS 静态渲染不受 JS 失败影响 | ✅ 已受控 |
| D3 | CDN 文件被误删导致 55 页面 404 | L | H | git 版本控制 + 分支隔离策略防止误删合并到 main | ✅ 已受控 |
| D4 | CSS 文件过大导致渲染阻塞 | L | L | 4 个 CSS/JS 文件总计 ~28 KB，本地加载 <10ms | ✅ 无影响 |

### E — Elevation of Privilege (权限提升)

| # | 威胁 | 可能性 | 影响 | 缓解 | 状态 |
|---|------|--------|------|------|------|
| E1 | YrY.copyCmd 被恶意页面调用覆盖剪贴板 | L | L | 需用户点击触发（事件驱动），非自动执行；剪贴板内容在用户下次粘贴时可见 | ✅ 已受控 |
| E2 | YrY.clipboardWrite 静默写入剪贴板 | L | L | 仅响应调用（需页面 JS 显式调用），非自动触发；HTTPS 环境要求 | ✅ 已受控 |
| E3 | CSS 注入通过 var() 引用未定义变量 | L | L | 浏览器忽略无效 var() 引用，不会执行代码 | ✅ 平台防护 |
| E4 | 页面内联 JS 通过 YrY 方法操纵 DOM 跨页面 | L | L | 同源策略限制；页面间不共享 DOM | ✅ 平台防护 |

## XSS 防护审查

| API | 方法 | 防 XSS | 评价 |
|-----|------|--------|------|
| YrY.toast() | `el.textContent = msg` | ✅ | textContent 自动转义所有 HTML 标签 |
| YrY.esc() | 替换 `& < > "` | ⚠️ | 不完整 — 缺少单引号 `'` 转义。在属性值场景中使用 `'` 包裹时存在注入风险 |

> 证据: `cdn/shared.js:74–77` — YrY.esc 仅转义 4 个字符

**建议**: YrY.esc() 补充单引号转义（`'` → `&#39;`），达到 OWASP XSS 防护规则 #1 的完整覆盖。

## 依赖安全

| 依赖 | 类型 | 风险 | 评价 |
|------|------|------|------|
| Google Fonts (fonts.googleapis.com) | 外部 CSS | 隐私泄露 + 可用性 | ⚠️ Category A 页面依赖外部 CDN。建议提供自托管字体回退方案 |
| navigator.clipboard API | 浏览器 API | 兼容性 | ⚠️ 非 HTTPS 环境不可用。内部页面通常以 `file://` 协议打开，clipboard API 可能被阻止 |
| cytoscape (unpkg CDN) | 知识图谱页面 | 可用性 + 完整性 | 仅知识图谱.html 使用，非 shared.js 依赖 |

## 审计结论

**整体风险等级: 低**

yry-cdn 作为静态前端资源库，威胁面有限：
- ✅ 不处理用户认证/授权
- ✅ 不存储或传输敏感数据
- ✅ 不涉及服务端逻辑
- ✅ 文件通过 git 版本控制管理，变更可追溯
- ⚠️ Google Fonts 外部依赖存在隐私/可用性风险（影响 Cat A 页面）
- ⚠️ YrY.esc() 缺少单引号转义（低风险，属性值场景）

**STRIDE 覆盖: 6/6 完成**
