📋 6 步迁移流程

  • ① 分析页内 <style> 对比 CDN 覆盖清单 — 找出可替换块
  • ② 判定页面类型 (Cat A 架构图/知识图谱 vs Cat B 审查/测试/演示/计划)
  • ③ 添加 <link> + <script> CDN 引用 — shared.css + theme.css/mono + shared.js
  • ④ 替换 CSS 类名为 yry-* 前缀 — .container→.yry-container 等
  • ⑤ 替换 JS 函数为 YrY.* 调用 — toast()→YrY.toast() 等
  • ⑥ 保留页面专属样式 — 删除 CDN 已覆盖块，仅留特有样式

📊 迁移效果估算 (单页面)

  • 📏 文件大小: 33.9 KB → ~18 KB (减少 ~47%)
  • 🎨 内联 CSS: ~300 行 → ~80 行 (减少 ~73%)
  • ⚡ JS 函数: 3 重复 → 0 (使用 YrY.*)
  • 📦 CDN 引用: 0 → 3 (shared.css + theme.css + shared.js)
