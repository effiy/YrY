加载顺序约束

  • ① fonts.css 最先 — @font-face 声明使浏览器尽早开始字体下载
  • ② shared.css 先于主题 — CSS Reset 必须在组件样式之前应用
  • ③ 主题 CSS 在 shared.js 前 — JS 操纵 .yry-* 类名需 CSS 已定义
  • ④ shared.js 最后 — 事件委托绑定在 DOM + CSS 就绪后
