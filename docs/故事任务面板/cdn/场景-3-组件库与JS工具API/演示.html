YrY API 分类架构

3 交互 API toast / copyCmd / switchPanel
·
3 Suite API initSuiteToggle / expandAll / collapseAll
·
3 工具函数 fmtDur / esc / clipboardWrite
📐 设计原则
• IIFE 封装 — 私有变量不污染全局命名空间
• 懒初始化 — DOM 容器（toast 元素）仅在首次调用时创建
• 事件委托 — Suite toggle 绑定在父容器上，支持动态内容
• 防竞态 — toast 使用 clearTimeout 确保最新消息不被旧定时器清除
