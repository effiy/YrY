| TC-UI-E3 | API 服务不可用 | 打开故事面板 | 显示网络错误提示；提供重试能力；不白屏 | |

| TC-UI-E5 | 组件 JS 路径不存在 | 初始化视图 | 显示错误状态；指明失败组件；不崩溃 | |
| TC-UI-E6 | 远端 API 延迟 > 5min | Web UI 发请求 | 显示超时错误；请求被中止；不阻塞其他操作 | |
