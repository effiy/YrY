  2. §2.2 发布命令 — git tag vX.Y.Z → git push --tags → npm publish
  3. §2.3 发布后验证 — npm 安装 · jsDelivr · unpkg · SRI 哈希
  4. §2.4 files 白名单策略 — 发布消费者所需资源 · 排除开发配置
  5. §2.5 发布流水线 (CI/CD) — PR → CI → 版本推断 → CHANGELOG → tag → npm publish → canary → latest
  6. §2.6 版本策略决策树 — BREAKING → MAJOR · feat → MINOR · fix → PATCH

  • npm 安装成功 (npm install yry-cdn)
  • jsDelivr 可访问 (HTTP 200)
  • unpkg 可访问 (HTTP 200)
  • 包体积 < 5MB (当前 ~3MB)
  • 版本号正确递增
  • CHANGELOG.md 含当前版本
  • Git tag 存在
  • 5 步加载链可用 (≤ 625ms)
