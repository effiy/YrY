加载链由 5 个不可交换顺序的步骤组成，每步对应 jsDelivr URL 下的一个固定资源。CSS 必须先于 JS，Vue 运行时必须先于组件 JS，shared/index.js 必须先于组件 JS 以提供 YrY.* 9 个工具 API。

🚀 性能优化策略

  • 字体预加载 — <link rel="preload" as="font" type="font/woff2" crossorigin>，FOUT 消除
  • DNS 预解析 — <link rel="dns-prefetch" href="https://unpkg.com">，DNS 耗时 -50ms
  • 资源预加载 — <link rel="preload" href="vue.global.prod.js" as="script" crossorigin>，Vue 加载提前
  • 动画延迟 — animation-delay: 100ms，避免首帧卡顿
  • 5s 超时容错 — fetch + AbortController，单点失败不阻塞
  • 组件按序加载 — 基础组件 (tag-chip → item-card → card-grid) 先于复合组件

🛡️ 容错降级

失败场景降级策略用户体验
Vue 3 CDN 不可达静态 HTML 渲染内容可读，交互不可用
组件 CSS 加载失败浏览器默认样式内容可读，布局降级
组件 JS 加载失败槽位显示占位符页面结构完整
字体加载失败系统等宽字体回退功能不受影响
