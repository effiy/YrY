① 5 步加载链清单 (不可交换顺序)

1shared/index.css — Reset + 动画 + CSS 变量 · <link> 同步 · ~15ms
2theme/index.css — Cat B System 主题 · <link> 同步 · ~10ms
3组件 CSS (107 个) — 按需组件样式 · <link> 按类别 · ~80ms
4Vue 3 运行时 — vue.global.prod.js · <script> 同步 · ~120ms
5组件 JS — Custom Elements 注册 · <script> 按序 · ~400ms

② 6 项性能优化策略

1字体预加载 — <link rel="preload" as="font" type="font/woff2" crossorigin> · FOUT 消除
2DNS 预解析 — <link rel="dns-prefetch" href="https://unpkg.com"> · DNS 耗时 -50ms
3资源预加载 — <link rel="preload" as="script" crossorigin> · Vue 加载提前
4动画延迟 — animation-delay: 100ms · 避免首帧卡顿
55s 超时容错 — fetch + AbortController · 单点失败不阻塞
6组件按序加载 — tag-chip → item-card → card-grid 先于 doc-layer

③ 4 项容错降级

1Vue 3 CDN 不可达 → 静态 HTML 渲染 (内容可读 · 交互不可用)
2组件 CSS 加载失败 → 浏览器默认样式 (内容可读 · 布局降级)
3组件 JS 加载失败 → 槽位显示占位符 (页面结构完整)
4字体加载失败 → 系统等宽字体回退 (功能不受影响)

④ 性能预算

1HTML 体积 ≤ 30KB (当前 18KB) · 关键 CSS ≤ 14KB (当前 9KB)
2JS 总量 gzip ≤ 80KB (当前 62KB) · 字体 woff2 ≤ 100KB (当前 88KB)
3FCP ≤ 310ms · TTI ≤ 480ms · 总加载 ≤ 625ms (均边界)
4首屏请求 ≤ 20 (当前 14) · 缓存命中率 ≥ 95%
