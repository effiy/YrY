🎯 实施步骤

  1. §2.1 最小页面骨架 — 引入 5 类资源 (shared CSS · theme CSS · Vue · shared JS · 组件 JS)
  2. §2.2 加载顺序约束 — CSS → Vue → shared → 组件 JS · 基础组件先于复合组件 · defer 非关键 JS
  3. §2.3 容错降级 — Vue 不可达→静态 HTML · CSS 失败→默认样式 · JS 失败→占位符 · 字体失败→系统等宽
  4. §2.4 SRI 与 CDN 回退 — sha384 哈希 · jsDelivr → unpkg → 自托管三级回退 · CSP 白名单
  5. §2.5 HTTP/2 与关键路径 — 多路复用 · 103 Early Hints · Brotli · 关键 CSS 内联 · 按需加载

✅ 验收清单

  • 5 步加载链全部 HTTP 200 (DevTools Network)
  • FCP ≤ 310ms (Lighthouse)
  • TTI ≤ 480ms (Lighthouse)
  • 总加载 ≤ 625ms (PerformanceObserver)
  • FOUT 不可见 (WebPageTest filmstrip)
  • CDN 断开后内容可读 (手动测试)
  • CSP 无 unsafe-inline (CSP Evaluator)
  • SRI 完整性 0 警告 (Browser Console)
  • HTTP/2 协议生效 (Response Header)
  • Brotli 压缩生效 (curl -sI)
  • CDN 缓存命中率 ≥ 95% (CDN 仪表盘)
