💡 演示 2 · Fallback 兜底

在 DevTools 中临时禁用 theme/index.css，观察组件回退到 fallback 值：

/* 禁用 theme/index.css 后 */
.yry-card {
  background: var(--yry-surface, rgba(15,23,42,.55));    /* ← 使用 fallback */
  border: var(--yry-border, 1px solid rgba(255,255,255,.06));
  color: var(--yry-text1, #a9b1d6);
}

预期：组件仍可读，视觉与启用 theme 时略有差异但布局不变。

💡 演示 3 · 运行时主题切换

在浏览器控制台执行：

// 切换到 Mono 主题
document.documentElement.setAttribute('data-theme', 'mono');
localStorage.yryTheme = 'mono';

// 切换回 System 主题
document.documentElement.setAttribute('data-theme', 'system');
localStorage.yryTheme = 'system';

// 跟随系统偏好
document.documentElement.removeAttribute('data-theme');

预期：整页瞬间变色，无 FOUC、无 JS 重新加载、组件状态保留。
