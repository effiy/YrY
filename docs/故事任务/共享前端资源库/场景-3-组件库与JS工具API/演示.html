💡 演示 1 · 组件注册与就绪事件

在浏览器 Console 执行：

// 检查组件是否已注册
console.log(customElements.get('yry-item-card'));   // 构造器或 undefined

// 监听就绪事件
document.addEventListener('yry-item-card-ready', function () {
  console.log('yry-item-card 已注册，API 可调用');
});

// 检查 9 个 API 可用性
Object.keys(window.YrY).forEach(function (k) {
  console.log(k, typeof window.YrY[k]);
});

💡 演示 2 · 双设策略时序

模拟元素出现早于组件注册的场景：

// 1. 元素已插入 DOM, 组件尚未注册 (此时 props 不会生效)
document.body.innerHTML +=
  '<yry-item-card id="late-card"></yry-item-card>';
var el = document.getElementById('late-card');
el.name = '晚到的组件';
el.icon = '📦';

// 2. 组件 JS 加载完成 (模拟 fetch + defineCustomElement)
setTimeout(function () {
  // ... 加载组件 JS
  document.dispatchEvent(new CustomEvent('yry-item-card-ready'));
  // ready 事件触发后, 组件接收到了之前的 props 赋值
}, 500);

💡 演示 3 · YrY.* API 实战

// toast 通知
YrY.toast('保存成功', 'success', 2000);
YrY.toast('网络错误', 'error', 3000);

// copyCmd 剪贴板
YrY.copyCmd('npm install yry-cdn');

// sceneMeta 7 交付物链接
var links = YrY.sceneMeta('../cdn/yry-arch/scenes/场景-1-新人上手');
console.log(links);
// [
//   { icon: '📋', label: '清单', href: '.../计划清单.html' },
//   { icon: '📐', label: '架构', href: '.../架构图.html' },
//   { icon: '🔗', label: '图谱', href: '.../知识图谱.html' },
//   { icon: '🧪', label: '测试', href: '.../测试面板.html' },
//   { icon: '📄', label: '源码', href: '.../源码.html' },
//   { icon: '💡', label: '演示', href: '.../演示.html' },
//   { icon: '📝', label: '审查', href: '.../审查.html' }
// ]

// switchPanel 切换浮动面板
YrY.switchPanel('cron');      // 打开定时任务面板
YrY.switchPanel('notify');    // 打开通知面板

// fetchData 带超时 + 重试
YrY.fetchData('/api/health', { timeout: 5000 })
  .then(function (r) { return r.json(); })
  .catch(function (err) { console.error('fetch 失败:', err); });

// ready Promise
YrY.ready(document.getElementById('card'), 'yry-item-card-ready')
  .then(function (el) { console.log('组件就绪:', el); });

💡 演示 4 · 错误降级场景

  • 断网 — fetchData 3 次重试后 reject，调用方 catch 处理
  • 非 https — copyCmd 自动退回 document.execCommand('copy')
  • 面板不存在 — switchPanel('unknown') 静默降级 + 日志
  • 元素未注册 — ready(el, 'yry-x-ready') 5s 超时 reject
  • DOM 未就绪 — toast 降级到 console.warn
