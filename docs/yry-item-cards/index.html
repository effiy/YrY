🃏 YryItemCards · 静态资产卡片数据注入器

自定义元素 <yry-item-cards> — 无渲染,纯数据+逻辑组件。
加载后自动遍历页面 <yry-card-grid id="..."> 元素并设置 .items 属性,
同时通过 MutationObserver / yry-layer-*-ready 事件 / 周期轮询三层兜底确保 yry-layer-agents/rules/refs 异步渲染后能挂载数据。
