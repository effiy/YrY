## 故事任务面板结构

每个故事目录包含：

```
故事任务面板/<name>/
├── 故事任务.md              # 问题空间基线（Story# · FP# · AC# · SC# · 风险）
├── 场景-N-<slug>.md         # 场景层，自包含 §0-§4 全生命周期
│   ├── §0 技术评审           # 效果示意 · 数据流 · 安全考量
│   ├── §1 测试设计           # 测试用例 · Gate A 交接
│   ├── §2 实施报告           # 操作步骤 · 源码清单 · P0 审查
│   ├── §3 测试报告           # 执行摘要 · 用例详情 · 失败分析
│   └── §4 自改进             # D0-D7 诊断 · 改进清单 · 评审清单
├── 场景-N-<slug>.html        # 架构图（自包含 HTML+SVG）
├── plan.html                 # 故事级计划总览
├── 知识图谱.json              # 结构化知识表示
├── 知识图谱.html              # 知识图谱可视化
├── .memory/                  # 执行记忆
└── .improvement/             # 改进提案
```
