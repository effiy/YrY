📐 四维加权评估模型

组件质量指数 CQI = CSS规范性 × 30% + API一致性 × 25% + 可访问性 × 20% + 响应式 × 25%

① CSS 规范性(30%) — 使用设计令牌变量(CSS var())·无硬编码颜色·BEM 命名规范·选择器深度≤3·无!important 滥用·文件大小≤5KB。检查标准：stylelint + arch-check。
② API 一致性(25%) — 命名空间统一(yry-*)·属性/方法签名稳定·事件命名规范(panel-hub-select 等)·返回值类型统一·版本向后兼容。检查标准：cdn-summary.json 接口定义。
③ 可访问性(20%) — ARIA 属性完备·键盘可达(Tab 导航)·屏幕阅读器友好·色彩对比度≥4.5:1·焦点管理·触控目标≥44px。检查标准：a11y 自动化检查。
④ 响应式(25%) — 移动端适配(≤720px)·弹性布局(flex/grid)·断点策略明确·触控友好·内容不溢出。检查标准：响应式测试矩阵。
