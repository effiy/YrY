sequenceDiagram
    participant P as Page HTML
    participant C as CSS (head)
    participant V as Vue (head)
    participant L as Loader (body)
    participant M as Mount Script (body)
    participant R as nav.breadcrumb

    P->>C: link css 加载
    P->>V: script vue 加载
    V-->>P: window.Vue 就绪
    P->>L: script loader 加载
    L->>L: fetch template → parse → register
    L-->>P: window.YryBreadcrumb 就绪
    L-->>P: dispatchEvent yry-breadcrumb-ready
    P->>M: mountBreadcrumb()
    M->>V: Vue.createApp(YryBreadcrumb, props)
    V->>R: mount → render nav
    R-->>P: 面包屑可见

📐 4 文件加载顺序

  1. <link rel="stylesheet" href="yry-breadcrumb/index.css"> — CSS 先加载防 FOUC
  2. <script src="vue.global.prod.js"> — Vue 3 运行时
  3. <script src="yry-breadcrumb/index.js"> — Loader 注册组件 + 派发 ready
  4. mountBreadcrumb() 内联脚本 — 创建 Vue app + 挂载
