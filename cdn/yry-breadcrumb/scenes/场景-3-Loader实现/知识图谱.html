⚙️ Loader (index.js) 核心编排器
YryBreadcrumb 异步加载器，IIFE 自执行格式，通过 document.currentScript 自定位推导模板路径，统一编排 fetch → DOMParser → 注册 → ready 事件全流程。
📐 架构图 📄 源码
🌐 fetch + DOMParser 数据获取与解析
fetch(index.html) 以 same-origin 凭证获取模板 HTML 文本；DOMParser.parseFromString 将文本转为 DOM Document；getElementById 提取 yry-breadcrumb-tpl 模板节点。每步均有 catch 错误捕获。
📐 架构图 🧪 测试
📡 ready 事件 事件机制
CustomEvent('yry-breadcrumb-ready') + dispatchEvent 实现加载与挂载的解耦通信。页面通过 {once:true} 监听器接收信号后执行 Vue.createApp mount，无轮询检查。
📐 架构图 💡 演示
🛡️ 超时保护 防御策略
setTimeout 5000ms + timedOut 标志，防止网络异常时无限等待。超时后 console.error 记录日志、不注册组件、不抛异常，页面其余内容正常显示 — 静默降级策略。

⚙️
Loader
核心模块
YryBreadcrumb 异步加载器 (index.js)，通过 document.currentScript 定位脚本 URL 推导模板路径，统一编排 fetch → 解析 → 注册 → 通知全流程。
→ fetch · DOMParser · ready ← 4 文件引用链
🌐
fetch
api
fetch(index.html) 获取模板 HTML 文本，作为 Loader 核心流程的第一步，失败时触发降级策略。
→ DOMParser ← Loader
📡
ready 事件
事件机制
CustomEvent('yry-breadcrumb-ready') + dispatchEvent，配合 {once:true} 监听器实现加载与挂载的解耦通信。
→ Vue.createApp mount ← 组件注册完成
🛡️
超时保护
防御
setTimeout 5s + timedOut 标志，防止网络异常时无限等待。超时后不注册组件，页面其余内容正常显示。
→ 降级策略 ← Loader 防御层
🔄
降级策略
策略
任何加载/解析/注册失败时静默降级 — console.error 记录日志、不注册组件、不抛异常，确保页面其余功能正常。
→ 页面继续运行 ← 超时 · fetch 失败
📋
4 文件引用顺序
约束
CSS → CSS → CSS → JS 严格加载顺序：shared/index.css → theme/index.css → index.css → index.js，确保样式优先、无 FOUC 闪烁。
→ Loader 执行 ← shared/index.css 起始
