① 5 步页面集成加载模式

1CSS 最先 — <link rel="stylesheet" href="yry-breadcrumb/index.css"> · 防 FOUC
2Vue 3 运行时 — <script src="../shared/vue.global.prod.js"> · CDN
3挂载点 — <div id="breadcrumb-app"></div> · 空容器
4Loader — <script src="yry-breadcrumb/index.js"> · fetch 模板 + 注册组件 + 派发 ready
5mount 脚本 — 内联 <script> · 守卫 + items 配置 + createApp

② 4 种 items 配置模式

1href + icon — {href, label, icon} · 完整可点击链接 + 图标 + 文字
2纯文本链接 — {href, label} · 仅可点击文字链接
3长路径 (回溯) — 5+ 项的多级面包屑 · 末项渲染为 aria-current="page"
4图标混用 — 部分项含图标 · 部分项纯文字 · v-if 条件渲染

③ mount 脚本守卫 (双设策略)

// 第 1 次: 立即赋值 (元素已在 DOM, 组件可能未定义)
var bc = document.getElementById('breadcrumb-app');
if (bc) {
  bc.items = [/* ... */];
}

// 第 2 次: 监听 ready 事件 (覆盖组件晚于元素出现的场景)
document.addEventListener('yry-breadcrumb-ready', function () {
  if (bc) bc.items = [/* ... */];
});

守卫逻辑：先检查 bc 存在 · 再监听 ready · 两次赋值确保数据一定被组件接收。

④ dogfooding 自举验证

129 个页面集成 — 计划清单 + 场景 1-5 (共 29 页) 全部引用 <yry-breadcrumb>
2FOUC 防护 — CSS 在 <head> 最先加载 · 无样式闪烁不可见
3多实例隔离 — 每页 1 个实例 · props 独立 · 互不干扰
4降级验证 — 断网 / Vue 缺失场景下页面仍可读 · 仅无交互
