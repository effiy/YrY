① 6 个设计目标清单

1零硬编码颜色 — CSS 变量 var(--yry-color-*, #fallback) · 切换主题后颜色自动适配
2a11y 语义化 — aria-label / aria-current / aria-hidden · 屏幕阅读器正确朗读
33 种数据模式 — v-if 条件渲染 · href+icon / 纯文本 / 回溯路径
45 种视觉模式 — 设计令牌 + 媒体查询 + 组件级覆盖 · 暗色/亮色/高对比/自定义/组件级
5响应式适配 — text-overflow: ellipsis + 视口断点 · 移动端不溢出
6BEM 命名 — .breadcrumb / .bc-sep / .bc-current · 低特异性 · 无样式冲突

③ 3 种数据模式判定

1href + icon — item.href 存在 + item.icon 存在 · 渲染 <a href> 含图标 + 文字
2纯文本链接 — item.href 存在 + item.icon 不存在 · 渲染 <a href> 仅文字
3当前项 (回溯路径末项) — item.href 不存在 · 渲染 <span aria-current="page"> 仅文字
