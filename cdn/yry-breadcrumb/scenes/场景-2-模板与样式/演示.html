自测

1 YryBreadcrumb 的 CSS 类名采用 BEM 命名规范，正确的类名是？
AYryBreadcrumb、YryBreadcrumbItem、YryBreadcrumbLink
B.breadcrumb、.breadcrumb__item、.breadcrumb__link
C.bc-container、.bc-item、.bc-link
✓ 正确! 采用 BEM 命名：Block 为 .breadcrumb，Element 为 .breadcrumb__item 和 .breadcrumb__link，Modifier 为 .breadcrumb__link--current 标记当前项。
2 组件 CSS 设计令牌使用什么语法引用，如何实现降级？
A@token(--yry-accent) + @supports 检测
Bvar(--yry-accent, #fallback) 内联降级值
Cenv(--yry-accent) 环境变量回退
✓ 正确! 所有颜色通过 var(--yry-*, #fallback) 引用，第二个参数为降级值。无自定义属性环境自动使用硬编码默认深色主题。
3 5 种视觉模式中，自定义主题模式的实现方式是？
A修改 index.css 源码中的默认值
B通过 JavaScript 动态注入内联样式
C宿主页面在 :root 中重新定义 --yry-* 令牌值
✓ 正确! 宿主页面通过在 :root 中定义自定义属性（如 --yry-accent、--yry-bg 等）覆盖组件默认令牌，无需修改组件源码即可应用自定义主题。
