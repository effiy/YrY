组件结构

📐 核心文件 (3)
T
index.html模板源
<script type="text/x-template" id="yry-breadcrumb-tpl"> 块的唯一定义点。
设计 / 前端 改这里 → index.js 启动时自动 fetch 同步。
唯一来源 Vue template a11y
~ 30 行 template · 8 行 demo · × 3 mount 点
L
index.jsLoader
IIFE 加载器: document.currentScript.src 解析自身 URL → fetch() 拉取 index.html → DOMParser 提取模板 → Vue.createApp().mount() → 派发 yry-breadcrumb-ready 事件。
运行时入口 fetch DOMParser 5s 超时
~ 5 KB · no-bundler · CustomEvent
S
index.css样式
使用 --yry-color-* 设计令牌 + fallback。link href 引入 → 全局作用域 .breadcrumb / .bc-sep / .bc-current。修改不影响模板与逻辑。
设计令牌 fallback 全局类
~ 1 KB · BEM 命名 · --yry-*
