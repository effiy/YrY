💬 YryTipBox · 提示框组件

自定义元素 <yry-tip-box> — 多类型提示/警告/信息框。 通过 type 属性切换样式：default (金色) · success (绿色) · info (青色) · warn (红色)。 适用于文档中的重点提示、警告、最佳实践标注。

📋 使用方式

使用 <yry-tip-box type="default|success|info|warn">，内容直接放入。 默认类型有金色左边框和淡金色背景，success 绿色，info 青色，warn 红色。 <strong> 文本自动着色匹配提示类型。
