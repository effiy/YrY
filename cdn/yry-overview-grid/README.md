# YryOverviewGrid · 概览网格组件

> Vanilla 组件 · 自定义元素 `<yry-overview-grid>` · 概览信息网格

## 文件

```
yry-overview-grid/
├── index.html    # 模板源 + Demo 预览
├── index.js      # 网格交互逻辑
└── index.css     # 组件样式 (3KB CSS)
```

## 功能

- 概览信息网格布局
- 使用 `--yry-accent-rgb` 令牌

## 使用

```html
<link rel="stylesheet" href="../../../../cdn/yry-overview-grid/index.css">
<script src="../../../../cdn/yry-overview-grid/index.js"></script>
<yry-overview-grid></yry-overview-grid>
```

## 依赖

无外部依赖，纯 Web Component
