# YryInlineHelpers · 内联工具类

> 内联工具类: 文字色 · 背景色 · 间距等 demo 辅助样式

## 文件

```
yry-inline-helpers/
├── index.html    # 工具类预览页
└── index.css     # 内联工具类 (12KB CSS)
```

## 状态

⚠️ **待完善** — 缺少独立 JS 交互层

## 功能

- 文字颜色工具类 (`.yry-text-*`)
- 背景色工具类 (`.yry-bg-*`)
- 间距辅助类 (margin/padding)
- Demo 页面快速原型工具

## 加载

```html
<link rel="stylesheet" href="../../../../cdn/yry-inline-helpers/index.css">
```

## 设计意图

提供类似 Tailwind 的工具类,用于 Demo 页面和快速原型开发,避免在 Demo 中写内联样式。
