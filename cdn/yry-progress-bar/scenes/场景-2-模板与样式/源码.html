YryProgressBar · 场景 2/5 · 模板 HTML · CSS 关键规则 · 动画关键帧

Vue 模板 (index.html)

<script type="text/x-template" id="yry-progress-bar-tpl">
  <div class="pb-wrap"
       role="progressbar"
       :aria-label="label || '进度'"
       :aria-valuenow="pct" aria-valuemin="0" aria-valuemax="100">
    <div class="pb-track">
      <div class="pb-fill"
           :class="{ 'is-done': pct >= 100, 'is-empty': pct <= 0 }"
           :style="{ width: pct + '%' }">
        <div class="pb-shine"></div>
      </div>
    </div>
  </div>
</script>

主机样式 — sticky + 玻璃 (index.css)

:root {
  --yry-progress-bar-height: 21px;
}
yry-progress-bar {
  display: block;
  position: sticky;  top: 0;  z-index: 100;
  background: linear-gradient(180deg,
    rgba(22,22,32,.96) 0%, rgba(22,22,32,.78) 100%);
  -webkit-backdrop-filter: blur(14px) saturate(160%);
  backdrop-filter: blur(14px) saturate(160%);
  border-bottom: 1px solid rgba(255,255,255,.08);
  padding: 6px 0 8px;
}

扫光动画 (index.css)

.pb-shine {
  position: absolute;
  width: 45%;
  background: linear-gradient(90deg,
    transparent 0%, rgba(255,255,255,.38) 50%, transparent 100%);
  animation: pb-shine 2.6s ease-in-out infinite;
}
@keyframes pb-shine {
  0%   { transform: translateX(-100%); }
  55%  { transform: translateX(220%); }
  100% { transform: translateX(220%); }
}
