📖 故事总览

🌐 故事 1: CDN 共享前端资源库

YrY 共享前端资源 — 双主题系统 (Mono + System)、107 组件、9 JS 工具 API、54 Vue 3 自定义元素。55+ 页面统一引用，消除内联重复代码 40-60%。覆盖从资源加载、主题设计、组件库 API、存量迁移到 npm 发布的全生命周期。

场景 1: CDN 资源加载与页面渲染
jsDelivr URL 规范 · 加载顺序约束 · 字体/动画延迟 · shared/index.css 必备基线
场景 2: 双主题系统设计
Category A (Mono) vs Category B (System) 选型 · 14 设计令牌 · CSS 变量分层
场景 3: 组件库与 JS 工具 API
107 组件（10 大类别）· 9 个 YrY.* 工具函数 · 54 个 Vue 3 自定义元素
场景 4: 存量页面迁移
删除内联样式 → 替换 .yry-* 类名 · 保留页面专属 CSS · 6 步迁移清单
场景 5: npm 包发布与版本管理
package.json 字段 · files 白名单 · jsDelivr 同步 · 版本号策略

📐 故事 2: YryBreadcrumb 组件

YryBreadcrumb 组件的独立故事 — 跟踪 5 个组件演化场景，从需求到发布的全生命周期。零打包 Vue 3 组件，3 文件拆分 (index.html/js/css)，异步 fetch + DOMParser 架构。

场景 1: 需求与设计
5 项设计决策 · Props 草案 · a11y 完备性约束
场景 2: 模板与样式
Vue template + CSS 设计令牌 · 3 个 Demo 示例
场景 3: Loader 实现
fetch + DOMParser + ready 事件 + 5s 超时容错
场景 4: 页面集成
4 文件引用顺序 · 4 种 items 模式 · 异步挂载
场景 5: 测试与发布
7 项自测 · npm 集成 · jsDelivr 同步验证
