说明

上图展示场景 2 · 模板与样式 的三层架构与数据流向。

Template 层

v-for 指令遍历 items 数组生成面包屑项；v-if 控制分隔符 '/' 的插入时机 (i > 0) 及链接/当前项的切换；:class 动态绑定当前页样式；aria-label 与 aria-current 保证可访问性。

Style 层

所有颜色值通过 --yry-color-* CSS 自定义属性驱动，支持 prefers-color-scheme: light 自动切换；每个 var() 均附带硬编码后备值，确保令牌不可用时仍可渲染。BEM 命名 .bc-sep / .bc-current 保持低特异性，易于覆盖。

Loader 层

index.js 异步加载 Vue 3 并通过 createApp 挂载组件；当组件定义就绪时派发 yry-breadcrumb-ready 自定义事件，页面监听此事件后执行挂载，实现解耦加载。

颜色 #ffc107 为本场景主色 (amber/gold)，与场景 tag 一致。
