4 文件加载顺序

顺序文件作用依赖
1shared/index.css定义 :root 设计令牌 + 基础重置无
2theme/index.css深色/浅色主题变量覆盖shared/index.css
3index.css面包屑 BEM 样式 + 令牌引用shared/index.css
4index.jsLoader 脚本 + Vue 组件注册 + ready 事件Vue 3 CDN、shared.js
📄
Template 域
template
Vue 指令集合 — v-for 迭代 items、v-if 条件渲染、:class 动态绑定、:key 列表稳定、aria-* 无障碍语义标注，全部驱动 DOM 生成。
→ index.html ← Props API (场景 1)
🎨
CSS 域
css
设计令牌体系 — --yry-color-* / --yry-radius-* / --yry-dur-* 统一管理色值、圆角、动效时长，BEM 命名 .bc-sep / .bc-current 保持低特异性，所有 var() 引用均带 fallback 降级。
→ index.css ← shared/index.css · theme/index.css
🔗
Integration 域
integration
4 文件加载链 — index.html (模板定义) → index.css (样式注入) → index.js (运行时挂载) → shared.js (共享工具)，构成完整渲染管线。
→ #breadcrumb-app ← Template · CSS · Shared
🎯
渲染输出
output
最终挂载到 #breadcrumb-app 的 DOM 节点，prefers-color-scheme 双模式适配，5 种视觉模式覆盖全部使用场景。
→ 黑暗/明亮双主题 ← Integration 域
♿
a11y 语义
accessibility
通过 aria-label 标注导航角色、aria-current="page" 标识当前页、BC-sep aria-hidden 隐藏分隔符装饰，确保屏幕阅读器可理解。
→ 屏幕阅读器 ← Template 域
图谱洞察 — 场景 2 展示 Template / CSS / Integration 三域分离的架构：模板域定义结构语义，CSS 域管理视觉令牌，集成域串联加载顺序。三者通过 index.html → CSS → JS 的严格加载链汇聚到最终 DOM 输出。所有设计令牌均配备 var() fallback 降级策略，确保跨主题兼容。
