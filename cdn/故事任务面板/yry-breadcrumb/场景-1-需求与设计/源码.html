index.html — 模板片段

Vue 3 模板源，定义面包屑的 DOM 结构。含 aria-label、aria-current、aria-hidden 可访问性标注。

HTMLindex.html
<script type="text/x-template" id="yry-breadcrumb-tpl">
  <nav class="breadcrumb" :aria-label="ariaLabel">
    <template v-for="(item, i) in items" :key="i">
      <span v-if="i > 0" class="bc-sep" aria-hidden="true">/</span>
      <a v-if="item.href" :href="item.href">
        <template v-if="item.icon">{{ item.icon }}&nbsp;</template>{{ item.label }}
      </a>
      <span v-else class="bc-current" :aria-current="i === items.length - 1 ? 'page' : null">
        <template v-if="item.icon">{{ item.icon }}&nbsp;</template>{{ item.label }}
      </span>
    </template>
  </nav>
</script>

index.js — 核心逻辑

异步加载流程：fetch 获取 index.html → DOMParser 提取模板 → Vue.createApp 挂载。

JSindex.js
// 异步加载模板并注册全局组件
(async function init() {
  try {
    // 1. 确定脚本所在目录，拼接 index.html 路径
    const scriptSrc = document.currentScript.src;
    const base = scriptSrc.substring(0, scriptSrc.lastIndexOf('/'));
    const htmlUrl = base + '/index.html';

    // 2. 异步 fetch HTML，通过 DOMParser 提取模板
    const resp = await fetch(htmlUrl);
    const text = await resp.text();
    const parser = new DOMParser();
    const doc = parser.parseFromString(text, 'text/html');
    const tpl = doc.querySelector('#yry-breadcrumb-tpl');

    // 3. 注册为全局 Vue 组件
    window.YryBreadcrumb = {
      template: tpl ? tpl.outerHTML : '<nav class="breadcrumb"></nav>',
      props: {
        items: { type: Array, default: () => [] },
        ariaLabel: { type: String, default: '面包屑导航' },
        separator: { type: String, default: '/' }
      }
    };

    // 4. 通知等待中的页面
    window.dispatchEvent(new CustomEvent('yry-breadcrumb-ready'));
  } catch (e) {
    console.warn('YryBreadcrumb: 模板加载失败，使用降级渲染', e);
  }
})();

index.css — 设计令牌使用

样式层使用 CSS 自定义属性 var(--yry-*, #fallback) 实现跨主题可移植，无自定义属性时降级为默认深色主题。

CSSindex.css
/* 设计令牌驱动：所有颜色通过 var(--yry-*, fallback) 引用 */
.breadcrumb {
  display: flex; flex-wrap: wrap; align-items: center; gap: 4px;
  padding: 6px 0; font-size: .82rem; line-height: 1.5;
  color: var(--yry-color-text-2, #d4d4d4);
}
.breadcrumb a {
  color: var(--yry-color-text-2, #d4d4d4); text-decoration: none;
  border-radius: 4px; padding: 2px 6px;
  transition: color .15s, background .15s;
}
.breadcrumb a:hover {
  color: var(--yry-color-accent, #ffc107);
  background: rgba(255,193,7,.08);
}
.bc-sep {
  color: var(--yry-color-text-3, #888); user-select: none;
}
.bc-current {
  color: var(--yry-color-accent, #ffc107); font-weight: 600;
}
