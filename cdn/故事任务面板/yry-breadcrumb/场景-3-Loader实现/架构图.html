sequenceDiagram
    participant Browser as 浏览器
    participant IndexJS as index.js
    participant Network as Network
    participant IndexHTML as index.html
    participant DOMP as DOMParser
    participant Vue as Vue.createApp

    Browser->>IndexJS: <script src="index.js">
    IndexJS->>IndexJS: document.currentScript.src
    IndexJS->>IndexJS: setTimeout(5s) 超时计时器
    IndexJS->>Network: fetch(index.html)
    Network-->>IndexJS: HTML text
    IndexJS->>DOMP: parseFromString(htmlText, 'text/html')
    DOMP-->>IndexJS: Document
    IndexJS->>IndexJS: getElementById('yry-breadcrumb-tpl')
    IndexJS->>IndexJS: window.YryBreadcrumb = {...}
    IndexJS->>Vue: dispatchEvent('yry-breadcrumb-ready')
    Vue->>Vue: mount(breadcrumb-app)

关键流程说明

步骤参与者操作错误处理
1Browser加载 index.js (IIFE 自执行)Vue 缺失时 warn 并 return
2index.jsdocument.currentScript 定位脚本 URL—
3index.jsnew URL('index.html', scriptUrl) 推导模板路径—
4index.jssetTimeout 5000ms 超时保护超时时设 timedOut=true，console.error
5Networkfetch(templateUrl, {credentials:'same-origin'})catch → console.error 不注册
6DOMParserparseFromString(htmlText, 'text/html')解析失败抛异常，由 catch 捕获
7index.jsgetElementById('yry-breadcrumb-tpl')id 缺失时 tpl 为 null
8VuecreateApp + mount('#breadcrumb-app')由页面 mountBreadcrumb 函数处理
