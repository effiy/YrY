自测

1 TC-5.1 使用什么命令验证 index.js 语法正确性？
Anpm test
Bnode --check index.js
Ceslint index.js
✓ 正确! node --check 是 Node.js 内置语法检查命令，不执行代码仅验证语法，exit code 0 表示无语法错误，适用于 IIFE/'use strict'/模板字符串等现代 JS 语法验证。
2 YryBreadcrumb 从 v1.0.0 升级到 v1.1.0 属于语义化版本的哪种变更？
AMAJOR — 不兼容的 API 修改
BMINOR — 向下兼容的功能新增
CPATCH — 向下兼容的问题修正
✓ 正确! 语义化版本 MAJOR.MINOR.PATCH 中，中间位 MINOR 版本号变更表示向下兼容的功能新增。v1.0.0→v1.1.0 新增了 a11y/异步加载/设计令牌等向下兼容特性。
3 package.json 中用于控制 npm publish 包含哪些文件的字段是？
Aexports
Bmain
Cfiles
✓ 正确! package.json 的 files 字段是一个数组，指定发布时包含的文件路径。YryBreadcrumb 的 files[] 包含 index.html、index.css、index.js、README.md 共 4 个路径。
