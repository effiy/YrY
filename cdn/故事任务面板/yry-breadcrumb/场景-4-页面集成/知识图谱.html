📐 4 文件引用顺序 加载管线
CSS → Vue → Loader → Mount 的严格加载顺序，不可重排。link css 防 FOUC 闪烁，script vue 提供运行时，script loader 注册组件并派发 ready，mount script 创建 Vue app 挂载实例。
📐 架构图 📄 源码
📦 4 种 items 模式 数据模式
href+icon / 纯文本 / 长路径 5+ / 图标混用 — 4 种实际使用模式覆盖全部页面场景的数据变体，确保组件在不同场景中的一致渲染。
💡 演示 🧪 测试
🔌 mount 脚本 挂载模式
Vue.createApp(YryBreadcrumb, props).mount('#breadcrumb-app') + yry-breadcrumb-ready 事件监听的解耦挂载模式。页面监听 ready 事件 (once:true)，收到后执行挂载。
