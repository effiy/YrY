💡 4 种 items 模式演示

下方 4 个 demo-block 分别表示 4 种 items 配置模式。每块独立挂载一个 YryBreadcrumb 实例。

模式 1 href + icon — 带链接的项，含 emoji 图标

模式 2 纯文本 — 所有项无 href/icon，当前页标记

模式 3 长路径 — 5+ 层级深度回溯

模式 4 图标混用 — 多个不同 emoji 图标 + href 混合

Mount 代码

JS
function mountDemo(id, items) {
  Vue.createApp(window.YryBreadcrumb, { items }).mount('#' + id);
}
function mountAll() {
  // 模式 1: href + icon
  mountDemo('demo-1', [
    { label: '文档中心', href: '../../../index.html', icon: '📄' },
    { label: '组件库' },
    { label: '导航类' },
    { label: '面包屑', icon: '🍞' }
  ]);
  // 模式 2: 纯文本
  mountDemo('demo-2', [
    { label: 'YrY' },
    { label: '故事面板' },
    { label: '场景 4' },
    { label: '页面集成' }
  ]);
  // 模式 3: 长路径 6 层
  mountDemo('demo-3', [
    { label: '首页', href: '/', icon: '🏠' },
    { label: '文档中心', href: '/docs' },
    { label: '故事面板', href: '/docs/stories' },
    { label: '计划清单' },
    { label: '场景 1 · 模板架构' },
    { label: 'CSS 设计系统', icon: '🎨' }
  ]);
  // 模式 4: 图标混用
  mountDemo('demo-4', [
    { label: 'YryBreadcrumb', href: '../../index.html', icon: '🍞' },
    { label: '任务故事', icon: '📋' },
    { label: '场景 4', icon: '🔌' },
    { label: '演示', icon: '🎮' }
  ]);
}
if (window.YryBreadcrumb) mountAll();
else document.addEventListener('yry-breadcrumb-ready', mountAll, { once: true });

AHref模式、Text模式、List模式、Grid模式
Bhref+icon、纯文本、长路径(5+层级)、图标混用
C链接模式、段落模式、层级模式、混合模式
