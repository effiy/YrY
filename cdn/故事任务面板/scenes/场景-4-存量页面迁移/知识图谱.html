🔗 内联样式 → CDN 组件映射图

[迁移前: 内联样式]                [迁移后: CDN 组件]
─────────────────────             ──────────────────────
<style>                            <link rel="stylesheet" href="yry-item-card/index.css">
  .card { ... }                   <link rel="stylesheet" href="yry-tag-chip/index.css">
  .card h3 { ... }                <script src="yry-item-card/index.js"></script>
  .card p { ... }                 <script src="yry-tag-chip/index.js"></script>
  .card .tag { ... }
</style>
                                  <yry-item-card
<div class="card">                  icon="📋"
  <h3>标题</h3>                  name="标题"
  <p>内容</p>                    badge="Vue 3"
  <div class="tags">              desc="内容"
    <span class="tag">标签</span>  tags='[{"text":"标签","modifier":"accent"}]'
  </div>                          meta="元信息">
</div>                            </yry-item-card>

迁移图谱：内联 <style> 块替换为 CDN <link>/<script> 引用；内联 HTML 结构替换为 <yry-*> 自定义元素；内联样式数据通过 property 传入组件。

📁 页面专属 CSS 保留场景

场景示例原因
页面特有布局特定网格列数 (如 5 列相册)组件不提供通用网格
页面特有动画首屏入场动画序列组件应保持动画无关
视觉细节特定间距微调组件间距由设计令牌控制
临时样式A/B 测试变体不应污染组件样式

⚠ 原则：页面专属 CSS 仅处理本页独有的样式，可复用模式必须提取为组件。

📊 内联消除度量方法

# 度量迁移前后的内联样式行数
grep -c '<style>' page.html           # style 块数
awk '/<style>/,/<\/style>/' file | wc -l  # 总行数
# 消除率 = (前 - 后) / 前 × 100%
指标度量方式目标
<style> 块数grep -c '<style>'≤ 1 (页面专属)
内联 style 行数awk 区间统计≤ 50 行
硬编码颜色grep -E '#[0-9a-f]{3,8}'≤ 0 (组件内)
重复类名grep -o '\.yry-' | sort -u≤ 1 处定义
组件引用率<yry-*> 标签 / 总节点≥ 30%
