⚠️ 风险与改进项

  • P1 · 性能余量紧 — FCP/TTI/总加载均贴近预算上限，组件 CSS 按需加载 + 关键 CSS 内联可释放 ~100ms
  • P1 · SRI 未启用 — 生产环境应生成 sha384 哈希并附加到所有 <script>/<link>
  • P2 · CDN 回退单点 — 当前仅 jsDelivr，未补 unpkg 作为第二回退源
  • P2 · 缓存策略 — 无 Service Worker，离线不可用；应上预缓存关键资源
  • P3 · 资源压缩 — 部分响应未带 content-encoding: br，CDN 边缘节点需启用 Brotli
