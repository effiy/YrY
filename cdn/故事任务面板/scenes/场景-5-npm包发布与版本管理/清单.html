① 7 步发布路径清单

1开发 — git commit · 实现新功能或修复
2构建 — npm pack --dry-run · 验证 files 白名单
3测试 — vitest + 视觉回归全部通过
4更新 CHANGELOG — 新增版本段落 · 列出变更项
5Tag — git tag vX.Y.Z · git push --tags
6发布 — npm publish · jsDelivr 自动同步 (< 5min)
7验证 — npm 安装 · jsDelivr/unpkg HTTP 200 · SRI 哈希

② package.json 关键字段

1name = yry-cdn · version = 1.2.0
2main = shared/index.js · style = shared/index.css
3unpkg = shared/index.js · jsdelivr = shared/index.css
4files = ["shared/","theme/","theme-mono/","fonts/","yry-*/","*.md"]

③ 3 段 SemVer 策略

1MAJOR (x.0.0) — 破坏性变更 · 移除组件/重命名令牌 · 回滚 30 天
2MINOR (x.y.0) — 向后兼容新增 · 新组件/新 API · 回滚 14 天
3PATCH (x.y.z) — Bug 修复 · 性能/文档 · 回滚 7 天
