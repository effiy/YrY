# YryTestPage · 测试页面组件

> Vanilla 组件 · 自定义元素 `<yry-test-page>` · 18KB CSS · 测试页面模板

## 文件

```
yry-test-page/
├── index.html    # 模板源 + Demo 预览
├── index.js      # 测试页面逻辑
└── index.css     # 组件样式 (18KB CSS)
```

## 功能

- 测试页面布局和样式
- 测试结果展示
- 使用 30+ CSS 令牌

## 使用

```html
<link rel="stylesheet" href="../../../../cdn/yry-test-page/index.css">
<script src="../../../../cdn/yry-test-page/index.js"></script>
<yry-test-page></yry-test-page>
```

## 依赖

无外部依赖，纯 Web Component
