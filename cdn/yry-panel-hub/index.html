🛠️ YryPanelHub · Vue 3 浮动面板工具栏

浮动面板管理工具栏 — label(可选) + 多个 button(icon/name/desc) + flow(可选)。
点击事件: 组件在根元素上派发 CustomEvent('panel-hub-select', { detail: { panel } }),页面监听后调用自己的面板 API。
模板源见下方 <script type="text/x-template" id="yry-panel-hub-tpl"> 块。
