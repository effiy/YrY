📝 YryCodeBlock · 代码块组件

自定义元素 <yry-code-block> — 带语言标签和复制按钮的代码块。 通过 lang 属性设置语言标签，copy 属性控制复制按钮。 Light DOM 内容为代码文本，自动保留空格和换行。
