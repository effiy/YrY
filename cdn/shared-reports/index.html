🔧 YrYReports API 参考

API签名用途
scoreClr(score: number) → string分数 → CSS 颜色变量名
scoreGrade(score: number) → string分数 → 字母等级 A/B/C/D
fmtMinutesAgo(min: number) → string"5 分钟前" / "2 小时前" / "3 天前"
renderFreshnessBadge(date: Date) → string新鲜度 badge HTML
renderScoreSparkline(scores: number[]) → stringSVG 迷你趋势图
trendDelta(curr: number, prev: number) → object趋势方向 + delta 值
statCard(val, label, cls?) → string统计卡片 HTML
badge(text, variant?) → string彩色 badge HTML
fetchCrossRefData() → Promise跨仪表板并行抓取数据

📊 统计函数

函数说明
mean(arr)算术平均值
stddev(arr)标准差
cv(arr)变异系数 (stddev / mean)
median(arr)中位数
linearSlope(arr)线性回归斜率 (趋势方向)

🎨 CSS 工具类

shared-reports/index.css 提供报告页面的统一样式:

  • .sr-card — 报告卡片容器
  • .sr-stat-grid — 统计网格布局
  • .sr-dim-bar — 维度评分进度条
  • .sr-trend-indicator — 趋势指示器 (↑/↓/→)
  • .sr-badge-{a,b,c,d} — A/B/C/D 等级徽章
  • .sr-sparkline — 迷你趋势图容器
  • .sr-freshness — 数据新鲜度指示器
