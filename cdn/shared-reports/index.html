YrYReports API

YrYReports.scoreClr(s)             // 分数 → CSS 颜色变量
YrYReports.scoreGrade(s)           // 分数 → 字母等级 A/B/C/D
YrYReports.fmtMinutesAgo(min)      // "5 分钟前" / "2 小时前" / "3 天前"
YrYReports.renderFreshnessBadge(d) // 新鲜度 badge HTML
YrYReports.renderScoreSparkline(scores) // SVG 迷你图
YrYReports.trendDelta(curr, prev)  // 趋势方向 + delta
YrYReports.statCard(val, label, cls) // stat 卡片 HTML
YrYReports.badge(text, variant)    // 彩色 badge HTML
YrYReports.fetchCrossRefData()     // 跨仪表板并行抓取
YrYReports.mean(arr) / stddev / cv / median / linearSlope
YrYReports.setText/setHTML/toggle
