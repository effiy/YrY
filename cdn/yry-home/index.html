设计令牌别名映射

yry-home 定义了 --color-accent → --yry-accent 等别名，使首页可以使用更简短、语义化的变量名。

--color-accent

→ --yry-accent (#FFC107)

--color-accent-2

→ --yry-cyan (#22d3ee)

--text-primary

→ --yry-text (#fafafc)

--text-secondary

→ --yry-text2 (#a0a0a4)

--text-muted

→ --yry-text3 (#6e6e72)

--bg-content

→ --yry-bg

--bg-card-flat

→ --yry-bg-flat

--border-subtle

→ --yry-border

--radius-card

→ --yry-radius (12px)

--shadow-card

→ --yry-shadow

--shadow-hover

→ --yry-shadow-lg

--cyan

→ --yry-cyan
