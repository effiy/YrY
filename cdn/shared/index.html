🌐 shared · 全局共享

CSSJS 提供 .yry-breadcrumb / .yry-cross-nav / .yry-toast / .yry-toolbar / .yry-footer 等通用 UI 元素 + 全局 YrY 命名空间工具方法。

载入

<link rel="stylesheet" href="../../../../cdn/shared/index.css">
<link rel="stylesheet" href="../../../../cdn/theme/index.css">
<script src="../../../../cdn/shared/index.js"></script>

CSS 主要类

  • .yry-breadcrumb · 面包屑导航
  • .yry-cross-nav · 同场景页面切换条
  • .yry-toast · 浮动提示（控制用 JS YrY.toast(msg)）
  • .yry-toolbar · 操作工具栏（含 .yry-toolbar-toggle / .yry-toolbar-actions）
  • .yry-footer · 页脚
  • .yry-kbd-hint · 键盘快捷键提示标签
  • .h-* · 健康报告相关组件（h-tabs / h-tab / h-panel / h-section / h-summary-card / h-dim-card / h-sparkline / h-struct-table / h-comp-table 等）

JS API（window.YrY）

YrY.toast('已保存')                // 浮动提示
YrY.copyCmd(btn, 'npm install')   // 一键复制 + 按钮态变化
YrY.switchPanel('walkthrough')    // 切 .yry-tab / .yry-panel
YrY.initTabs('.container')
YrY.initSuiteToggle('.yry-container')
YrY.expandAllSuites() / collapseAllSuites()
YrY.initDemoTabs(sceneKey, keyMap, onHelp)
YrY.initChecklist({storageKey, exportTitle})
YrY.initRadar(data, baseData, descMap, defaultMsg)
YrY.initNextAction(steps)
YrY.initCodeCopy('.container')
YrY.initStepToggle('.container')
YrY.fmtDur(1234)                   // → "1.2s"
YrY.esc(htmlString)
YrY.sceneMeta('./base-path')      // 7 件套链接数组
YrY.elt(tag, className, textContent)
