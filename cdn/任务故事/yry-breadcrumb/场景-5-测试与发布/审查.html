发布检查清单

发布前必须逐项确认以下内容，全部通过方可执行 npm publish。

  • 语法检查 — node --check cdn/yry-breadcrumb/index.js 通过，exit code 0
  • 打包验证 — npm pack --dry-run 确认 files[] 列表完整，含 4 个 yry-breadcrumb 路径
  • 包内容验证 — tar -tzf yry-cdn-1.1.0.tgz | grep yry-breadcrumb 列出 3 个核心文件 + README
  • 浏览器自测 — 直接打开 index.html，3 个 demo (href+icon / 纯文本 / 回溯路径) 全部渲染
  • 控制台清洁 — 无 Vue warning、fetch error、404 或 JS 异常
  • 键盘可达 — Tab 键可从第 1 个链接依次聚焦，current 项不可聚焦（符合规范）
  • 屏幕阅读器 — 朗读顺序为 link → (separator skip) → link → ... → current
  • 跨场景链接 — 场景 1-4 的任务故事链接均可正常跳转，无 404
  • npm publish 就绪 — 版本号 v1.1.0，package.json 字段齐全，npm whoami 已登录
