🔔 YryToast · Vue 3 Toast 通知组件

全局 Vue 3 组件 — 通过 window.YryToast.show/success/error/warn/info 调用。
懒加载挂载: 首次调用时才创建 Vue 应用并渲染,平时零开销。
5 种语义色 success / warn / error / info / default · 上限 5 条 · 自动消失 · 点击关闭。

Demo 2 — 触发多个叠加 (上限 5)
