📐 步骤
  1. 勾选进度联动: checkbox → updateProgress() → 进度条 width + overview 统计 + localStorage 持久化
  2. 折叠面板: 点击交付物头部 → CSS max-height 0.3s ease-out 过渡动画
  3. 标签页切换: 点击标签/键盘 1-9 → panel opacity 切换 + URL hash + localStorage tab 记忆
  4. 风险行展开: 点击风险行 → CSS class open toggle + 缓解步骤详情展开
  5. 交付物类型过滤: 全部/HTML/MD/CSS/Test/其他 六类 + 展开/折叠全部 + 一键复制路径
🔍 要点
零依赖内联 JS · 组件间通过 updateProgress() 收敛状态 · 键盘跳过 input 聚焦防误触发 · 交付物扩展名自动标注
