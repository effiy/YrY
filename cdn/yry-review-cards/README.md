# YryReviewCards · 审查卡片组

> Vanilla 组件 · 自定义元素 `<yry-review-cards>` · 审查结果卡片组

## 文件

```
yry-review-cards/
├── index.html    # 模板源 + Demo 预览
├── index.js      # 卡片交互逻辑
└── index.css     # 组件样式 (4KB CSS)
```

## 功能

- 审查结果卡片展示
- 3 种语义色 (pass / warn / fail)

## 使用

```html
<link rel="stylesheet" href="../../../../cdn/yry-review-cards/index.css">
<script src="../../../../cdn/yry-review-cards/index.js"></script>
<yry-review-cards></yry-review-cards>
```

## 依赖

无外部依赖，纯 Web Component
