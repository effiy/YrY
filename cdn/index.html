🧩 工具组件 (Vanilla JS · 零配置)

以下组件为纯 JavaScript 实现，不依赖 Vue。引入 CSS + JS 即可使用，无需手动初始化。

📋 yry-export-toolbar — 导出工具栏

支持 Copy PNG / Download PNG / Download PDF / Copy SVG / Download SVG 五种操作。 通过 data-target 指定截图目标，data-filename 设置文件名。

📸 截图目标区域

点击下方按钮导出此区域

🔗 yry-cytoscape-graph — 知识图谱

封装 Cytoscape.js，自带 YrY 深色主题、8 种节点类型 × 4 种边类型样式、邻域高亮交互、 图例自动生成。支持 data-elements-url 异步加载或 element.graphData 内联数据。
