YryBreadcrumb — Vue 3 面包屑组件

3 文件拆分 · 异步模板加载 · a11y 完备 · 5s 超时容错
🔌 加载顺序
1

2
<script src="https://unpkg.com/vue@3/dist/vue.global.prod.js">
Vue 3 运行时 — 全局 Vue.createApp API，供 index.js Loader 使用
3
<script src="cdn/yry-breadcrumb/index.js">
Loader 自动执行 — fetch 模板 → 注册 window.YryBreadcrumb → 派发 yry-breadcrumb-ready
🔧 Props 定义
I
items (必填)
Array<{label: string, href?: string, icon?: string}> — 有 href 渲染为 <a>，无 href 渲染为 <span aria-current="page">
A
ariaLabel (可选)
默认 '面包屑导航' — <nav> 的 aria-label 属性值
S
separator (可选)
默认 '/' — 分隔符文本，渲染为 aria-hidden="true" 的 <span>
🌐 使用示例
U
挂载代码
function mount() {
  Vue.createApp(window.YryBreadcrumb, {
    ariaLabel: '面包屑导航',
    items: [
      { label: '文档中心', href: 'index.html', icon: '🗃' },
      { label: '场景 1 · 模板架构' }
    ]
  }).mount('#breadcrumb-app');
}
window.YryBreadcrumb ? mount() : document.addEventListener('yry-breadcrumb-ready', mount, { once: true });
