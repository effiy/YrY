📊 CDN 评分体系 — 六维加权健康评估模型
CDN 资源库采用六维加权评分体系，覆盖组件质量(CQI)·综合健康(CHI)·测试质量(TQI)·自改进闭环(SII)·加载链稳定(LSI)·页面迁移(MPI) 六大维度。每维度独立评估、独立评级，不强制归一化。以下各卡片详述评估方法、权重分配与数据来源。

🧩 组件质量指数 CQI

四维加权评估 62 个 CDN 组件（10 大类 · 加权均分）。CSS 规范性(35%): BEM 命名规范·选择器深度≤3·设计令牌使用率·无 !important 滥用。API 一致性(30%): yry-* 命名空间统一·属性签名稳定·版本向后兼容。可访问性(20%): ARIA 属性完备·键盘可达·对比度≥4.5:1·触控目标≥44px。响应式(15%): ≤720px 适配·弹性布局·断点策略一致。

🩺 综合健康指数 CHI

六维加权聚合 → 组件覆盖度×0.30 (CSS/主题/API/脚本四类覆盖率) + 测试覆盖×0.25 (vitest 单元/集成/视觉回归三项均分) + 自改进闭环×0.20 (D0-D7诊断触发率·建议采纳率·问题收敛率) + 加载链稳定×0.15 (5步链路可靠性·容错降级·回退完备性) + 存量迁移率×0.10 (内联→CDN组件迁移完成度·55+页面) + 字体性能×0.05 (woff2预加载·font-display策略)。D0-D7八级诊断·总加载≈625ms·v1.0(78/C)→v1.1(86/B)→v1.2(92/A) 持续上升。

🧪 测试质量指数 TQI
