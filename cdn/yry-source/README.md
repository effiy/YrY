# YrySource · 源码页面样式

> Vanilla 组件 · 源码页面样式 + 弹窗管理 · 25KB CSS · 3KB JS

## 文件

```
yry-source/
├── index.html    # 源码样式预览
├── index.js      # 源码页面交互: 弹窗管理 (3KB JS)
└── index.css     # 源码页面专属样式 (25KB CSS)
```

## 功能

- 源码页面布局和样式
- 代码行号展示
- 弹窗管理 (代码片段放大)

## 加载

```html
<link rel="stylesheet" href="../../../../cdn/yry-source/index.css">
<script src="../../../../cdn/yry-source/index.js"></script>
```
