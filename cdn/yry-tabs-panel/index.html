📑 YryTabsPanel · 标签页 + 面板切换

纯 CSS + shared.js 组件 — 使用 data-panel 属性关联 tab 与 panel。
YrY.initTabs() 自动绑定点击事件，读取 data-panel 切换面板。

📊 总览 📋 详情 ⚙ 设置
📊 总览面板 — 这是默认显示的标签页内容。点击上方标签切换。
📋 详情面板 — 显示详细信息。使用 data-panel="details" 关联。
⚙ 设置面板 — 配置选项。使用 data-panel="settings" 关联。

💡 使用方式：
data-panel="name" 对应 id="panelName"（首字母大写）。
初始化调用 YrY.initTabs()，可选传入容器选择器。
