# rui-docs Rules

1. 操作范围仅限当前项目 `docs/` 目录，不得触及 `.claude/`、代码文件或其他目录
2. sync 为增量合并，不删除本地文件；冲突文件需用户手动裁决
3. retro 复盘文档写入根项目 `docs/自改进故事面板/`，格式为 `<project>-docs-<date>.md`
4. retro 仅分析本地 `docs/` 结构，不连接远端
5. 空输入不得自动执行管线，仅推荐任务
6. 禁止直接修改 `docs/` 下内容，所有改动必须从 main 拉取 `feat/<name>` 分支进行
7. 禁止自动合并功能分支到 main，合并操作一律由开发者手动执行
8. API_X_TOKEN 仅从系统环境变量读取，不写入代码、日志或配置
