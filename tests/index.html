🚀 运行测试

全量测试

运行所有 vitest 测试 (lib + skills)
npm test

全量测试 + 遗留测试

vitest + 遗留测试一起运行
npm run test:all

监视模式

文件变更时自动重新运行测试
npm run test:watch

覆盖率报告

生成 HTML/JSON/text 覆盖率报告到 coverage/ 目录
npm run test:coverage

UI 模式

在浏览器中打开 vitest UI 交互式查看测试结果
npm run test:ui
