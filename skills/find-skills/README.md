# find-skills 快速索引

`find-skills` 用于在 `.claude/skills/` 下发现并推荐可用技能。行为真源在 `SKILL.md`。

## 快速开始

```bash
/find-skills <任务描述> [关键词1,关键词2,...]
```

示例：

```bash
/find-skills "设计文档生成" "文档,E2E"
/find-skills "代码审查与安全检查" "安全,lint"
```

## 文件职责

| 文件           | 职责                           |
| -------------- | ------------------------------ |
| `SKILL.md`     | 匹配逻辑、置信度标准、输出格式 |

## 使用原则

1. 只返回 `.claude/skills/` 下真实存在的技能目录名，禁止编造。
2. 返回的技能名必须与目录名完全一致，不得做大小写或别名变换。
3. 若找不到匹配，输出"未找到合适技能"。
