## 检查项

### P0 - 必须通过

| 检查项 | 检查内容 | 验证方法 |
|---------|---------|---------|
| **用户故事表格存在** | 包含从需求文档提取的用户故事表格 | 检查是否有"用户故事与验收标准"章节和表格 |
| **用户故事表格完整** | 用户故事表格包含 4 列：用户故事、验收标准、过程生成文档、产出智能文档 | 检查表格列数是否为 4 |
| **优先级图标使用** | 用户故事使用 🔴🟡🟢 优先级图标 | 检查表格中是否包含优先级图标 |
| **功能分解图存在** | 包含功能分解图（Mermaid graph TB） | 检查是否有"功能分解图"章节和 Mermaid 图表 |
| **用户流程图存在** | 包含用户流程图（Mermaid flowchart TD） | 检查是否有"用户流程图"章节和 Mermaid 图表 |
| **功能流程图存在** | 包含功能流程图（Mermaid flowchart TD） | 检查是否有"功能流程图"章节和 Mermaid 图表 |

### P1 - 应该通过

| 检查项 | 检查内容 | 验证方法 |
|---------|---------|---------|
| **完整时序图存在** | 包含完整时序图（Mermaid sequenceDiagram） | 检查是否有"完整时序图"章节和 Mermaid 图表 |
| **功能详情章节存在** | 包含"功能详情"章节 | 检查是否有"功能详情"章节 |
| **验收标准章节存在** | 包含"验收标准"章节 | 检查是否有"验收标准"章节 |
| **验收标准分级** | 验收标准按 P0/P1/P2 分级组织 | 检查是否有 P0、P1、P2 子章节 |
| **验收标准可测试** | 验收标准描述清晰，可验证可测试 | 主观判断验收标准是否可测试 |
| **使用场景章节存在** | 包含"使用场景示例"章节 | 检查是否有"使用场景示例"章节 |
| **使用场景格式规范** | 使用场景采用"背景-操作-结果"三段式格式 | 检查使用场景是否有"**背景**："、"**操作**："、"**结果**：" |
| **Mermaid 图表有说明** | 每个 Mermaid 图表下方都有明确的说明文字 | 检查每个图表后是否有说明 |
| **核心价值章节存在** | 包含"核心价值"列表 | 检查功能概述中是否有核心价值 |

### P2 - 可以有

| 检查项 | 检查内容 | 验证方法 |
|---------|---------|---------|
| **Mermaid 节点样式规范** | 节点色值符合规范要求 | 检查是否使用 #ccffcc、#e1f5ff 等标准色值 |
| **关联文档链接** | 有关联文档链接（需求文档、设计文档、使用文档） | 检查元数据中是否有关联文档 |
| **功能点有 Mermaid 图表** | 功能详情中的功能点有辅助说明图表 | 检查功能详情中是否有 Mermaid 图表 |
