# doc-quality — 文档质量标准

> 文档不是副产品，而是真相来源。质量标准 = 可验证、可追溯、可执行。

[文档等级](#文档等级) · [内容标准](#内容标准) · [结构要求](#结构要求) · [审查清单](#审查清单) · [退化检测](#退化检测)

---

## 文档等级

| 等级 | 定义 | 标识 | 处理 |
|------|------|:---:|------|
| **A** | 已验证 — 可通过 Read/Grep/Glob 验证 | ✅ | 可直接引用 |
| **B** | 可推导 — 从 A 级推导一步 | ⚡ | 可引用，标注推导链 |
| **C** | 未验证 — 待补充证据 | ⚠️ | **必须显式标注**「待补充」 |
| **D** | 虚构 — 无来源的断言 | 🚫 | **禁止出现** |

## 内容标准

### 必须包含

| 文档类型 | 必含元素 | 验证方式 |
|---------|---------|---------|
| Agent .md | frontmatter + 工作循环 mermaid + 规则表 + Red Flags + 合理化速查表 | 逐节检查 |
| Skill SKILL.md | 触发条件 + 输入/输出 + 使用示例 + 退出码 | `grep "## 触发" SKILL.md` |
| Rule .md | 适用范围 + 门禁条件 + 反例 | `grep "反例\|Anti-pattern" rule.md` |
| 故事任务.md | FP# + AC + SC + 风险 + 知识图谱基线 | 每 FP# 对应 ≥1 AC |

### 禁止出现

| 禁止项 | 为什么 | 替代做法 |
|--------|--------|---------|
| TBD / TODO | 不完整 = 不可执行 | 写完整或标注 Level C |
| "类似 XXX" | 引用不可追溯 | 写独立完整描述 |
| 无来源的性能声明 | Level D | 附测量数据或标注 Level C |
| "应该" / "可能" | 不可证伪 | 用"必须"（P0）/ "建议"（P1）/ "可选"（P2） |
| 过期截图 | 误导 | 标注截图日期或使用 mermaid（可文本 diff） |

## 结构要求

### Agent 文件统一模版

```
---
name: <name>
description: <一句话>
tools: <Read/Grep/Glob/Bash/Edit/Write>
---

# <name> — <中文职责>

> <一句话角色定位>

[工作循环](#) · [触发](#) · [规则表](#) · [Red Flags](#) · [职责边界](#) · [生效标志](#)

## 工作循环       ← mermaid 流程图
## 触发           ← 谁/什么触发该 Agent
## 规则表         ← # | 规则 | 反例
## Red Flags      ← 暂停信号 + 合理化速查表
## 职责边界       ← 归我 / 不归我
## 生效标志       ← 达标标准 | 未达标处置
```

### Skill 文件统一模版

```
---
name: <skill-name>
description: <一句话>
user_invocable: true/false
lifecycle: default-pipeline | standalone
---

# <skill-name>

> <一句话定位>

[触发](#) · [输入/输出](#) · [模式](#) · [约束](#) · [交付](#)

## 触发
## 输入/输出
## 模式（如有多种）
## 约束
## 交付（如是写入命令）
```

### Rule 文件统一模版

```
# <rule-name> — <中文主题>

> <一句话原则>

[适用范围](#) · [门禁](#) · [反例](#)

## 适用范围
## 门禁条件
## 反例 / 常见误判
```

## 审查清单

| # | 检查项 | 通过条件 |
|---|--------|---------|
| 1 | 无 TBD/TODO | `grep -r "TBD\|TODO" docs/ skills/` 返回空 |
| 2 | 无 D 级断言 | 所有陈述有来源标注或显式标注 Level C |
| 3 | 版本行正确 | 文档版本与 plugin.json 一致 |
| 4 | 交叉引用有效 | 所有内部链接可达（`grep` 验证目标锚点存在） |
| 5 | Mermaid 图可渲染 | 图在 mermaid.live 或本地预览中正常显示 |
| 6 | Agent 模版完整 | 含 frontmatter + 工作循环 + 规则表 + Red Flags + 合理化速查表 |
| 7 | Skill 模版完整 | 含 frontmatter + 触发 + 输入/输出 + 约束 |
| 8 | Rule 模版完整 | 含适用范围 + 门禁 + 反例 |

## 退化检测

文档退化的三个信号，出现任一即触发审查：

| 信号 | 检测方式 | 处置 |
|------|---------|------|
| **版本漂移** | `grep` 版本号 vs plugin.json | 更新文档版本行 |
| **链接断裂** | 404 / 锚点不存在 | 修复或移除链接 |
| **内容过时** | 文档描述与 `git diff main` 不一致 | 运行 `rui-update` 刷新 |
