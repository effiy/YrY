# rui-skills 质量验证模型

> Agent 技能生态发现的质量验证规则，独立于实现细节。

## 五维评分体系

| 维度 | 权重 | 评估标准 | 数据来源 |
|------|:---:|---------|---------|
| **来源信誉** | 30% | GitHub stars ≥ 1000 (满分) / ≥ 100 (及格) / < 100 (警告) | GitHub API |
| **社区活跃度** | 25% | 安装量 ≥ 10K (满分) / ≥ 1K (及格) / < 1K (警告) | npm/skills.sh |
| **维护健康度** | 20% | 最近更新 ≤ 30 天 (满分) / ≤ 90 天 (及格) / > 180 天 (警告) | GitHub API |
| **依赖安全性** | 15% | 无已知 CVE (满分) / 低危 CVE (及格) / 中高危 CVE (拒绝) | npm audit |
| **兼容性** | 10% | 与现有技能无冲突 (满分) / 有可选冲突 (警告) / 有强制冲突 (拒绝) | 本地检查 |

## 评分等级

| 等级 | 分数 | 建议 |
|------|:---:|------|
| **A** | ≥ 80 | 推荐安装，无需警告 |
| **B** | ≥ 60 | 可以安装，标注注意事项 |
| **C** | ≥ 40 | 谨慎安装，明确告知风险 |
| **D** | < 40 | 强烈不建议安装，需用户明确确认 |

## 风险标注

| 风险类型 | 标识 | 示例 |
|---------|------|------|
| 低信誉 | Low Reputation | GitHub stars < 100 |
| 长期未维护 | Unmaintained | 最近更新 > 180 天 |
| 安全漏洞 | CVE Found | 存在中高危 CVE |
| 依赖冲突 | Dep Conflict | 与已安装技能共享依赖版本冲突 |

## 技能生命周期

| 阶段 | 操作 | 检查点 |
|------|------|--------|
| 发现 | `/rui-skills find` | 搜索覆盖度、结果相关性 |
| 评估 | 五维评分 + 风险标注 | 评分 ≥ 60、无阻断风险 |
| 安装 | `/rui-skills add` | 兼容性检查、安装成功 |
| 使用中 | 日常调用 | 功能正常、无性能退化 |
| 更新 | `/rui-skills update` | 变更日志审查、回归测试 |
| 卸载 | `/rui-skills remove` | 依赖检查、残留清理 |

## 核心规则

| # | 规则 | 设计理由 |
|---|------|---------|
| 1 | 安装前验证来源信誉 | 防止恶意或低质量技能包 |
| 2 | 低信誉来源需明确警告用户 | 知情决策 |
| 3 | 中高危 CVE 为阻断条件 | 供应链安全不可妥协 |
| 4 | 安装失败时提供诊断信息 | 不阻塞用户 |
| 5 | 不自动安装未经确认的技能包 | 尊重用户自主权 |
| 6 | 安装后验证技能可用性 | 确保安装质量 |
| 7 | 技能更新前审查变更日志 | 避免意外引入破坏性变更 |

## 评分计算

```
总分 = Σ(维度得分 × 权重) / 100

其中每个维度得分:
  - 满分标准 → 100 分
  - 及格标准 → 60 分
  - 未达及格标准 → 按比例计算: (实际值 / 及格标准) × 60
  - 阻断条件 (CVE/冲突) → 0 分，直接 D 级
```

## 阻断条件

| 条件 | 级别 | 行为 |
|------|:---:|------|
| 中高危 CVE | 阻断 | 拒绝安装，提示风险 |
| 强制依赖冲突 | 阻断 | 拒绝安装，提示冲突详情 |
| GitHub stars = 0 | 警告 | 强烈不建议，需明确确认 |
| 最近更新 > 365 天 | 警告 | 标记为已废弃 |

## 安装后验证

| 验证项 | 方法 | 不通过处置 |
|--------|------|-----------|
| 技能可加载 | `npx skills list` 含新技能 | 重新安装 |
| 功能冒烟测试 | 执行技能基础命令 | 报告失败，建议卸载 |
| 无冲突 | 其他技能功能正常 | 记录冲突，提示用户 |
| 配置文件完整 | 检查技能目录结构 | 报告缺失文件 |

## 边界场景

| 场景 | 处置 |
|------|------|
| 搜索无结果 | 告知用户，建议替代关键词或自建技能 |
| 同名技能已安装 | 提示已安装版本，询问是否覆盖 |
| 安装过程中断 | 清理残留文件，提示重试 |
| GitHub API 限流 | 使用缓存评分，标注 `stale` |
| 离线环境 | 仅检查本地缓存，提示联网后刷新 |
