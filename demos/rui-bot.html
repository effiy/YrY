通知流程

操作命令

发送通知
发送故事状态变更通知到企微群
node skills/rui-bot/send.mjs --story=user-login --status=done
仅记录不发送
追加通知日志但不发送 HTTP 请求
node skills/rui-bot/send.mjs --story=user-login --status=doing --no-send
多状态通知
支持 doing / done / blocked 三种状态
node skills/rui-bot/send.mjs --story=user-login --status=blocked
