/rui init
六步建立项目基线：detect → explore → generate → arch → setup → verify
/rui init
/rui doc 需求
拆需求为故事 + 生成文档基线，全程只读源码
/rui doc 用户登录功能
/rui code name
实现故事：Gate A → 逐模块 → Gate B → 自改进
/rui code user-login
/rui update name
增量更新，T1/T2/T3 自动裁剪管线
/rui update user-login --no-code
/rui yry
全自主自改进闭环，循环至收敛
/rui yry --depth 3
/rui version --up
自主判定版本号 → 更新文件 → commit → push
/rui version --up
