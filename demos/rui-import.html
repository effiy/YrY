📤 rui-import · 文档同步

本地文档 → 远程 API — 1-5 数字键切换 Tab · 悬停代码复制

🚶 演练 📊 概览 ⚡ 命令 📁 文件 🧪 测试
全量同步（最常用）
项目根全量扫描所有文件 → 上传到远端 API。并发 4，自动区分 created / overwritten。默认排除 .git / node_modules / dist / .claude-plugin。
node skills/rui-import/sync.mjs📋
先预览再同步（安全工作流）
Step 1：输出待上传文件清单，不发 HTTP 请求。确认无误后 Step 2：去掉 mode=list 实际同步。推荐每次同步前执行。
node skills/rui-import/sync.mjs mode=list📋
从远端拉取文档
远端 → 本地覆盖指定目录。适用于恢复丢失的本地文档或同步他人更新。需配合 dir= 指定目标目录。
node skills/rui-import/sync.mjs dir=docs/故事任务面板/ mode=pull📋
单目录同步
指定目录扫描 + 上传，路径以项目根计算相对路径。支持前缀注入 prefix=a,b。
node skills/rui-import/sync.mjs dir=docs/故事任务面板/user-login/📋
状态检测
空输入触发状态检测：检查 token 可用性、远端可达性、文件差异分析。输出推荐操作而不实际同步。
/rui-import📋

最常用
全量同步
项目根全量扫描 → 逐文件上传，并发 4。自动区分 created / overwritten，输出汇总。
node skills/rui-import/sync.mjs📋
安全预览
仅枚举不上传
输出待上传文件清单，不发 HTTP 请求。确认无误后去掉 mode=list 实际同步。
node skills/rui-import/sync.mjs mode=list📋
目录同步
指定目录上传
指定目录扫描 + 上传，路径以项目根计算相对路径，与远端一一对应。
node skills/rui-import/sync.mjs dir=docs/故事任务面板/📋
远端下载
mode=pull 拉取
远端 → 本地覆盖指定目录。需配合 dir= 使用，names= 可按文件名关键词过滤。
node skills/rui-import/sync.mjs dir=.claude/ mode=pull📋
单文件
单文件导入
导入单个文件，自动附加语义标签和故事面板 session 关联。
node skills/rui-import/sync.mjs file=docs/故事任务面板/.../故事任务.md📋
扩展名过滤
exts= 覆盖扩展名
默认全扩展名。可设置 exts=md,json,yaml 限制上传文件类型。
node skills/rui-import/sync.mjs exts=md,json📋
