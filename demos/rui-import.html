全量同步（最常用）
项目根全量扫描 .md + .claude/ 文件 → 上传到远端 API。并发 4，自动区分 created/overwritten。
node skills/rui-import/sync.mjs workspace=true📋
先预览再同步（安全工作流）
Step 1：输出待上传文件清单，不发请求。确认无误后 Step 2：实际同步。
node skills/rui-import/sync.mjs workspace=true mode=list📋
从远端拉取故事文档
远端 → 本地覆盖整个故事目录。适用于恢复丢失的本地文档或同步他人更新。
node skills/rui-import/sync.mjs dir=docs/故事任务面板/user-login/ mode=pull📋
从远端拉取 .claude/ 配置
远端 → 本地覆盖 .claude/ 全量配置。适用于跨设备同步开发环境配置。
node skills/rui-import/sync.mjs dir=.claude/ mode=pull📋
单文件导入
导入单个文件，自动附加语义标签和故事面板 session 关联。
node skills/rui-import/sync.mjs file=docs/故事任务面板/user-login/故事任务.md📋
